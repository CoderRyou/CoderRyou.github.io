## 日语动词变形规则一览表

### 动词分类

首先，根据变形规则，日语动词主要分为三类：

| 动词类别 | 特点 | 例子 |
| :--- | :--- | :--- |
| **一类动词** | 词尾为 `う` 段假名，又称“五段动词”。 | **書く**、**話す**、**泳ぐ**、**立つ**、**死ぬ**、**呼ぶ**、**飲む**、**買う** |
| **二类动词** | 词尾为 `る`，且 `る` 前的假名在 `い` 段或 `え` 段上，又称“一段动词”。 | **食べる**、**起きる**、**見る**、**寝る** |
| **三类动词** | 只有两个动词，不规则。 | **する**、**来る** |

---

### 动词变形规则表

| 变形类别 | 用法简述 | 一类动词 (五段) | 二类动词 (一段) | **する** | **来る** |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **ます形 (敬体)** | 礼貌、正式场合 | **词尾 → い段 + ます**<br>書**く** → 書**き**ます<br>立**つ** → 立**ち**ます<br>買**う** → 買**い**ます | **去る + ます**<br>食べ**る** → 食べます<br>寝**る** → 寝ます | **します** | **来ます (きます)** |
| **ない形 (否定)** | 表示否定，"不..." | **词尾 → あ段 + ない**<br>書**く** → 書**か**ない<br>**\*例外：** 词尾为`う`时，变为`わ`<br>買**う** → 買**わ**ない | **去る + ない**<br>食べ**る** → 食べ**ない** | **しない** | **来ない (こない)** |
| **て形 (中顿)** | 连接句子，表顺序、方式等 | **1. 词尾为`く、ぐ`**<br>   `く → いて`，`ぐ → いで`<br>   書**く** → 書**いて**<br>   泳**ぐ** → 泳**いで**<br>**2. 词尾为`う、つ、る`**<br>   `→ って`<br>   買**う** → 買**って**<br>   立**つ** → 立**って**<br>**3. 词尾为`ぬ、ぶ、む`**<br>   `→ んで`<br>   死**ぬ** → 死**んで**<br>   呼**ぶ** → 呼**んで**<br>**4. 词尾为`す`**<br>   `→ して`<br>   話**す** → 話**して**<br>**特例：** 行**く** → 行**って** | **去る + て**<br>食べ**る** → 食べ**て**<br>見**る** → 見**て** | **して** | **来て (きて)** |
| **た形 (过去)** | 过去式，完成体 | **规则与`て形`完全相同，只需将`て`换为`た`，`で`换为`だ`**<br>書く → 書い**た**<br>泳ぐ → 泳い**だ**<br>立つ → 立っ**た**<br>買う → 買っ**た** | **去る + た**<br>食べる → 食べ**た** | **した** | **来た (きた)** |
| **ば形 (条件)** | 表示假设条件，"如果..." | **词尾 → え段 + ば**<br>書**く** → 書**け**ば<br>話**す** → 話**せ**ば | **去る + れば**<br>食べ**る** → 食べ**れば** | **すれば** | **来れば (くれば)** |
| **意向形 (意志)** | 表达意图或提议，"让我们..." | **词尾 → お段 + う**<br>書**く** → 書**こ**う<br>話**す** → 話**そ**う | **去る + よう**<br>食べ**る** → 食べ**よう** | **しよう** | **来よう (こよう)** |
| **可能形** | 表示能力，"能...，会..." | **词尾 → え段 + る**<br>書**く** → 書**け**る<br>話**す** → 話**せ**る | **去る + られる**<br>食べ**る** → 食べ**られる** | **できる** | **来られる (こられる)** |
| **受身形 (被动)** | 表示被动，"被..." | **词尾 → あ段 + れる**<br>書**く** → 書**か**れる<br>**\*例外：** 词尾为`う`时，变为`わ`<br>買**う** → 買**わ**れる | **去る + られる**<br>食べ**る** → 食べ**られる** | **される** | **来られる (こられる)** |
| **使役形** | 表示使役，"让...做" | **词尾 → あ段 + せる**<br>書**く** → 書**か**せる<br>**\*例外：** 词尾为`う`时，变为`わ`<br>買**う** → 買**わ**せる | **去る + させる**<br>食べ**る** → 食べ**させる** | **させる** | **来させる (こさせる)** |
| **命令形** | 下达命令（语气强硬） | **词尾 → え段**<br>書**く** → 書**け**！<br>話**す** → 話**せ**！ | **去る + ろ/よ**<br>食べ**る** → 食べ**ろ**！ (口语)<br>食べ**よ**！ (书面) | **しろ／せよ** | **来い (こい)** |
| **禁止形** | 表示禁止，"不准..." | **字典形（原形） + な**<br>書く**な**！<br>食べる**な**！ | **するな** | **来るな (くるな)** |

### 补充说明

1. **二类动词的例外**：有些动词以`いる/える`结尾，但属于**一类动词**，如`帰る`、`知る`、`入る`、`走る`等，需要单独记忆。
2. **礼貌的命令形**：`て形` + `ください`（请...）是更常用的礼貌命令形式，例如`書いてください`（请写）。
3. **被动与使役被动**：使役形 + 被动形（`させられる`）可以构成使役被动形，表示“被迫做...”，如`食べさせられる`（被迫吃）。
4. **~たい形 (愿望)**：`ます形`去`ます` + `たい`，表示“想要...”，如`食べます` → `食べたい`（想吃）。
