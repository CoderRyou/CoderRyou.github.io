日语动词变形规则

Ryouhh2025/10/15日语日语

日语动词变形规则一览表

动词分类

首先，根据变形规则，日语动词主要分为三类：

动词类别特点例子
一类动词词尾为 う 段假名，又称“五段动词”。書く、話す、泳ぐ、立つ、死ぬ、呼ぶ、飲む、買う
二类动词词尾为 る，且 る 前的假名在 い 段或 え 段上，又称“一段动词”。食べる、起きる、見る、寝る
三类动词只有两个动词，不规则。する、来る

动词变形规则表

变形类别用法简述一类动词 (五段)二类动词 (一段)する来る
ます形 (敬体)礼貌、正式场合词尾 → い段 + ます
書く → 書きます
立つ → 立ちます
買う → 買います		去る + ます
食べる → 食べます
寝る → 寝ます		します来ます (きます)
ない形 (否定)表示否定，"不..."词尾 → あ段 + ない
書く → 書かない
*例外： 词尾为う时，变为わ
買う → 買わない		去る + ない
食べる → 食べない		しない来ない (こない)
て形 (中顿)连接句子，表顺序、方式等1. 词尾为く、ぐ
く → いて，ぐ → いで
書く → 書いて
泳ぐ → 泳いで
2. 词尾为う、つ、る
→ って
買う → 買って
立つ → 立って
3. 词尾为ぬ、ぶ、む
→ んで
死ぬ → 死んで
呼ぶ → 呼んで
4. 词尾为す
→ して
話す → 話して
特例： 行く → 行って		去る + て
食べる → 食べて
見る → 見て		して来て (きて)
た形 (过去)过去式，完成体规则与て形完全相同，只需将て换为た，で换为だ
書く → 書いた
泳ぐ → 泳いだ
立つ → 立った
買う → 買った		去る + た
食べる → 食べた		した来た (きた)
ば形 (条件)表示假设条件，"如果..."词尾 → え段 + ば
書く → 書けば
話す → 話せば		去る + れば
食べる → 食べれば		すれば来れば (くれば)
意向形 (意志)表达意图或提议，"让我们..."词尾 → お段 + う
書く → 書こう
話す → 話そう		去る + よう
食べる → 食べよう		しよう来よう (こよう)
可能形表示能力，"能...，会..."词尾 → え段 + る
書く → 書ける
話す → 話せる		去る + られる
食べる → 食べられる		できる来られる (こられる)
受身形 (被动)表示被动，"被..."词尾 → あ段 + れる
書く → 書かれる
*例外： 词尾为う时，变为わ
買う → 買われる		去る + られる
食べる → 食べられる		される来られる (こられる)
使役形表示使役，"让...做"词尾 → あ段 + せる
書く → 書かせる
*例外： 词尾为う时，变为わ
買う → 買わせる		去る + させる
食べる → 食べさせる		させる来させる (こさせる)
命令形下达命令（语气强硬）词尾 → え段
書く → 書け！
話す → 話せ！		去る + ろ/よ
食べる → 食べろ！ (口语)
食べよ！ (书面)		しろ／せよ来い (こい)
禁止形表示禁止，"不准..."字典形（原形） + な
書くな！
食べるな！		するな来るな (くるな)

补充说明

  1. 二类动词的例外：有些动词以いる/える结尾，但属于一类动词，如帰る、知る、入る、走る等，需要单独记忆。
  2. 礼貌的命令形：て形 + ください（请...）是更常用的礼貌命令形式，例如書いてください（请写）。
  3. 被动与使役被动：使役形 + 被动形（させられる）可以构成使役被动形，表示“被迫做...”，如食べさせられる（被迫吃）。
  4. ~たい形 (愿望)：ます形去ます + たい，表示“想要...”，如食べます → 食べたい（想吃）。
