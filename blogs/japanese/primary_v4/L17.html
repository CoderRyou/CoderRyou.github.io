会话 - 日本取材の成果

王：（做大概介绍后）以上が，ホームページの「『金星』の故郷を探る」で取り上げる記事の一例です。

大山：ええ，確かに。でも，それは日本国内でのことですよ。

王：いえ，そんな…たいしたことではありません。すでに日本で販売されている「金星」のカクテルは何種類かあるそうですね。その中から，若者に喜ばれそうなものを選んでいただけませんか。中国用のネーミングやラベルのデザインは，私どもで引き受けます。

课文 - 北京の顔

  北京のほぼ中心部にある景山公園は，市民の憩いの場である。休日には，若いカップルや家族連れ，旅行者でにぎわう。

  公園の周囲には「胡同」と呼ばれる狭い路地が無数に存在している。

  「胡同」を知るには，歴史を約700年前までさかのぼらなければならない。13世紀，元の時代に，敵から市民を守るため，石垣が作られ，その中に住宅が作られた。この住居を取り囲むように通っているのが「胡同」である。

  胡同は長い間，北京市民の大切な生活の場であった。路地には朝市が頻繁に開かれ，屋台が並ぶ。時には自転車置き場にもなる。胡同には役割によってさまざまな名前がつけられている。現在でも入り口に名前が書かれた看板がかかっている路地も多い。かつて金銭の取引所があった「銭市胡同」，刃物を研ぐ職人が暮らしていた「磨刀胡同」など，昔のままの名前も残っている。

  胡同が作られた当時，「四合院」と呼ばれる，中国の伝統的な家屋も作られた。東西南北の棟が，庭を中心に建てられている。最近では，この四合院をホテルやレストランに改造しているところもある。

  胡同は元の時代にはせいぜい29本だったが，明の時代には459本に，1980年代には1，300本を超えた。その長さから，北京の胡同をつなぐと「もう1つの万里の長城になる」とまでいわれたそうである。

  残念なことに，近代化が進み，胡同の数が急激に減ってきてしまった。ただ，その一方，古い建物を残そうとする動きもある。北京市では25か所の歴史文化保護区を指定して，町並みを保存することを決めた。新しいものと古いものが混在する街北京は，今変革の時を迎えている。

课文翻译 - 北京的面孔

  位于北京几乎中心位置的景山公园，是市民的休憩场所。休息日，这里挤满了年轻情侣、家庭游客和旅行者。

  公园周围存在着无数被称为“胡同”的狭窄小巷。

  要了解“胡同”，必须追溯到约700年前的历史。13世纪的元代，为了抵御敌人保护市民，建造了石墙，在其中建造了住宅。像围住这些住宅一样穿行而过的就是“胡同”。

  胡同在很长一段时间里，是北京市民重要的生活场所。小巷里经常开设早市，摆着摊床。有时也会成为自行车停放处。胡同根据其作用被起了各种各样的名字。即使在现在，入口处挂着写有名字牌子的胡同也很多。过去曾有金钱交易场所的“钱市胡同”、磨刀匠人居住的“磨刀胡同”等，保留着原貌的名字也留存了下来。

  在建造胡同的那个时期，被称为“四合院”的中国传统房屋也被建造出来。东西南北的房屋以庭院为中心建造。最近，也有地方把这些四合院改建成酒店或餐厅。

  胡同在元代最多只有29条，到明代变成了459条，到1980年代超过了1300条。据说，根据其长度，如果把北京的胡同连接起来，“就会成为另一条万里长城”。

  遗憾的是，随着现代化的发展，胡同的数量急剧减少了。不过，另一方面，也有试图保留古老建筑的动向。北京市指定了25处历史文化保护区，决定保护街道风貌。新与旧共存的北京城，现在正迎来变革的时刻。

最近では，この四合院をホテルやレストランに改造しているところもある。最近，也有地方把这些四合院改建成酒店或餐厅。1. 最近では
说明： “在最近”。
2. ～に改造する
说明： “改造成……”。
胡同は元の時代にはせいぜい29本だったが，明の時代には459本に，1980年代には1，300本を超えた。胡同在元代最多只有29条，到明代变成了459条，到1980年代超过了1300条。1. せいぜい～だった
说明： “最多只有……”。
2. ～に増えた
说明： 此处省略了谓语，上下文表示变化。
3. ～を超える
说明： “超过……”。
その長さから，北京の胡同をつなぐと「もう1つの万里の長城になる」とまでいわれたそうである。据说，根据其长度，如果把北京的胡同连接起来，“就会成为另一条万里长城”。1. ～から
说明： 表示依据，“根据……”。
2. ～と
说明： 表示条件，“一……就……”。
3. ～とまでいわれる
说明： “甚至被称为……”。
4. ～そうである
说明： 传闻助动词的书面语，“据说……”。
残念なことに，近代化が進み，胡同の数が急激に減ってきてしまった。遗憾的是，随着现代化的发展，胡同的数量急剧减少了。1. 残念なことに
说明： “遗憾的是……”。
2. ～が進む
说明： “……发展”。
3. ～てきてしまった
说明： “～てくる”（变化趋势）+“～てしまう”（彻底完成/遗憾），表示“已经……掉了”。
ただ，その一方，古い建物を残そうとする動きもある。不过，另一方面，也有试图保留古老建筑的动向。1. その一方
说明： “另一方面”。
2. ～を残そうとする
说明： “想要保留……”。“～ようとする”表示打算、试图。
北京市では25か所の歴史文化保護区を指定して，町並みを保存することを決めた。北京市指定了25处历史文化保护区，决定保护街道风貌。1. ～を指定して～を決めた
说明： て形表示动作先后。
2. ～ことを決めた
说明： “决定……”。
新しいものと古いものが混在する街北京は，今変革の時を迎えている。新与旧共存的北京城，现在正迎来变革的时刻。1. ～が混在する
说明： “……共存”，作定语。
2. ～を迎えている
说明： “迎来……”，持续体表示状态。
