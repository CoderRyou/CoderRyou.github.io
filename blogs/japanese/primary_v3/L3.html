会话 - 顔合わせ

（JC策划公司和龙虎酒业公司的成员就座，开始自我介绍）

李： JC企画上海支社の李秀麗です。今回の「金星」プロジェクトは，私をふくよんめいたんとう含めて4名のスタッフが担当いたします。

野田： はじめまして。野田理恵です。上海支社に来て，2年目です。主に営業を担当させていただきます。どうぞよろしくお願いします。

王： はじめまして。王風と申します。王は王様の王，風は風と書きます。広告やホームページのデザインを担当しております。どうぞよろしくお願いいたします。

李： このほかに中井という男性社員が加わります。

佐藤： よろしくお願いします。李さんも日本語が上手ですけど，王さんも上手ですね。

王： ありがとうございます。出身は浙江省の紹興ですが，大学時代，日本に留学していたんです。ですから，今回のプロジェクトにたいへん興味があります。

佐藤： 紹興ですか。10年前に，1度だけ行ったことがあります。

王： そうですか。

佐藤： ええ。わたしにとっては，初めての海外旅行でしたが，紹興で飲んだ紹興酒の味は今でも覚えていますよ。

会话翻译

（JC策划公司和龙虎酒业公司的成员就座，开始自我介绍）

李： 我是JC策划公司上海分公司的李秀丽。这次的“金星”项目，包括我在内由4名员工负责。

野田： 初次见面。我是野田理惠。来到上海分公司是第2年。主要负责营业工作。请多关照。

王： 初次见面。我叫王风。“王”是国王的王，“风”是风雨的风。负责广告和网页设计。请多关照。

李： 除此之外，还有一位叫中井的男员工加入。

佐藤： 请多关照。李小姐日语很好，王先生也很不错呢。

王： 谢谢。我老家是浙江省绍兴，大学时代曾在日本留学。所以对这次项目非常感兴趣。

佐藤： 绍兴啊？10年前我只去过一次。

王： 是吗？

佐藤： 是的。对我来说那是第一次出国旅行，但在绍兴喝的绍兴酒的味道至今还记得呢。

会话语法讲解

原文翻译语法解析
今回の「金星」プロジェクトは，私をふくむ4名のスタッフが担当いたします。这次的“金星”项目，包括我在内由4名员工负责。1. ～をふくむ
说明： “包含……”，他动词。
2. が担当いたします
说明： “いたします”是“する”的自谦语，表示谦逊地承担。
上海支社に来て，2年目です。来到上海分公司是第2年。1. ～て
说明： 表示动作先后顺序。
2. ～目
说明： 接尾词，表示顺序，“第……年”。
主に営業を担当させていただきます。主要负责营业工作。语法： させていただく
说明： 使役态与“ていただく”结合的自谦表达，意为“请允许我做……”，非常礼貌。
はじめまして。王風と申します。初次见面。我叫王风。语法： と申します
说明： “と言います”的自谦语，用于谦逊地介绍自己名字。
王は王様の王，風は風と書きます。“王”是国王的王，“风”是风雨的风。语法： ～は～と書く
说明： “～写作……”，用于解释汉字的写法。
広告やホームページのデザインを担当しております。负责广告和网页设计。语法： ～ておる
说明： “ている”的自谦语，“おります”比“います”语气更谦逊。
どうぞよろしくお願いいたします。请多关照。语法： お願いいたします
说明： “お願いします”的更谦逊形式。
このほかに中井という男性社員が加わります。除此之外，还有一位叫中井的男员工加入。1. ～という
说明： “叫做……的”，用于说明具体内容或名称。
2. が加わる
说明： 自动词，表示“加入、参加”。
李さんも日本語が上手ですけど，王さんも上手ですね。李小姐日语很好，王先生也很不错呢。1. ～も～も
说明： “既……也……”，表示并列。
2. ～ですけど
说明： “ですが”的口语形式，表轻微转折或话题铺垫。
3. ～ですね
说明： 表示确认、感叹或寻求共鸣。
出身は浙江省の紹興ですが，大学時代，日本に留学していたんです。我老家是浙江省绍兴，大学时代曾在日本留学。1. ～が
说明： 接续助词，此处表示顺接，引出后文。
2. ～ていたんです
说明： “～ていた”表示过去持续的状态；“んです（のです）”用于说明情况或解释原因。
ですから，今回のプロジェクトにたいへん興味があります。所以对这次项目非常感兴趣。语法： ～に興味がある
说明： “对……感兴趣”，助词“に”表示兴趣指向的对象。
10年前に，1度だけ行ったことがあります。10年前我只去过一次。语法： ～たことがある
说明： 表示曾经有过某种经历。
わたしにとっては，初めての海外旅行でしたが，紹興で飲んだ紹興酒の味は今でも覚えていますよ。对我来说那是第一次出国旅行，但在绍兴喝的绍兴酒的味道至今还记得呢。1. ～にとっては
说明： “对……来说”，提示评价的角度。
2. ～た～
说明： “飲んだ”是过去式作定语，修饰“紹興酒”。
3. ～ている
说明： “覚えている”表示记忆的持续状态。
4. よ
说明： 终助词，用于强调或提醒对方注意。

课文 - 名字

  中国語の「名字」は多くの場合，姓と名の両方を指しますが，日本語では家族の姓を「名字」といいます。

  すべての日本人が名字を持つようになったのは，明治時代になってからです。当時の政府が，すべての国民が名字を持つことを法律で決めました。それまで名字を持っていたのは武士や貴族だけでした。ですから，突然名字を持てといわれても，どんな名字を持てばいいのか分かりません。人々は慌てました。

  しかし，名字をつけないわけにはいきません。そこで，多くの人が，地名や地形から名字をつけました。日本の名字に，「木」「林」「山」「川」など自然に関係する漢字が多いのは，そのためです。家が谷の中にあるから「中谷」，近くに大きな杉の木があるから「大杉」とつけられた名字もたくさんあります。また，日本で最も多い「佐藤」や 「鈴木」は，昔の武士の名字や，地名からつけられた名字だといわれています。

  日本人の名字の種類は，30万近くあります。それでも世界の第2位です。第1位は多民族国家のアメリカで，その数は160万を超えています。一方，最も少ないのが韓国で3 0 0程度しかありません。また，世界でいちばん人数の多い姓は「李」で，約1億人いるそうです。

课文翻译

  中文的“名字”大多指姓和名两者，但在日语中，家族的姓氏称为“名字”。

  所有日本人都有姓氏，是到明治时代之后才开始的。当时的政府通过法律规定所有国民都必须拥有姓氏。在此之前，只有武士和贵族才有姓氏。因此，突然被要求必须有姓氏，人们也不知道该取什么姓氏才好。大家都很慌张。

  但是，又不能不起姓氏。于是，很多人从地名或地形来取姓氏。日本的姓氏中，“木”“林”“山”“川”等与自然相关的汉字很多，就是这个原因。因为家在谷中而取“中谷”，因为附近有大杉树而取“大杉”的姓氏也很多。另外，日本最多的“佐藤”和“铃木”，据说源自古代武士的姓氏或地名。

  日本人的姓氏种类有近30万种。即便如此仍是世界第二。第一位是多民族国家美国，数量超过160万。另一方面，最少的是韩国，只有300种左右。此外，世界上人数最多的姓氏是“李”，据说约有1亿人。

课文语法讲解

原文翻译语法解析
中国語の「名字」は多くの場合，姓と名の両方を指しますが，日本語では家族の姓を「名字」といいます。中文的“名字”大多指姓和名两者，但在日语中，家族的姓氏称为“名字”。1. ～は～を指す
说明： “指”是他动词，“を”提示宾语。
2. ～を～という
说明： “把……称为……”。
3. ～が
说明： 接续助词，表示转折对比。
すべての日本人が名字を持つようになったのは，明治時代になってからです。所有日本人都有姓氏，是到明治时代之后才开始的。1. ～ようになる
说明： 表示状态的变化，“变得……”。
2. ～のは～からです
说明： “之所以……是因为……”。“のは”将前半句名词化，“から”表示起点、原因。
当時の政府が，すべての国民が名字を持つことを法律で決めました。当时的政府通过法律规定所有国民都必须拥有姓氏。1. ～が～ことを決める
说明： “……决定……事”。“こと”是形式名词，指代前面整个子句的内容。
2. 法律で
说明： “で”表示方式、手段。
それまで名字を持っていたのは武士や貴族だけでした。在此之前，只有武士和贵族才有姓氏。语法： ～だけ
说明： 副助词，表示限定，“仅仅、只”。
ですから，突然名字を持てといわれても，どんな名字を持てばいいのか分かりません。因此，突然被要求必须有姓氏，人们也不知道该取什么姓氏才好。1. ～といわれる
说明： 被动形式“被说……”，此处“持て”是“持つ”的命令形。
2. ～ばいい
说明： 表示建议，“只要……就行”。
3. ～のか
说明： “か”前接疑问词，表示不确定的内容。“のか”使其名词化，作“分かりません”的宾语。
人々は慌てました。大家都很慌张。（无特殊语法，陈述过去状态）
しかし，名字をつけないわけにはいきません。但是，又不能不起姓氏。语法： ～ないわけにはいかない
说明： 双重否定句式，表示出于情理或义务“不能不……”、“必须……”。
そこで，多くの人が，地名や地形から名字をつけました。于是，很多人从地名或地形来取姓氏。语法： ～からつける
说明： “から”表示出处、来源。
日本の名字に，「木」「林」「山」「川」など自然に関係する漢字が多いのは，そのためです。日本的姓氏中，与自然相关的汉字很多，就是这个原因。语法： ～が多いのは～ためです
说明： “之所以……多，是因为……”。“のは”名词化主语小句，“そのため”指代前文原因。
家が谷の中にあるから「中谷」，近くに大きな杉の木があるから「大杉」とつけられた名字もたくさんあります。因为家在谷中而取“中谷”，因为附近有大杉树而取“大杉”的姓氏也很多。1. ～から
说明： 表示原因。
2. ～とつけられる
说明： 被动形式“被取名为……”。“と”表示称谓的内容。
また，日本で最も多い「佐藤」や 「鈴木」は，昔の武士の名字や，地名からつけられた名字だといわれています。另外，日本最多的“佐藤”和“铃木”，据说源自古代武士的姓氏或地名。1. ～からつけられる
说明： （同上）表示来源。
2. ～といわれている
说明： 表示普遍的传闻或说法，“据说……”。
日本人の名字の種類は，30万近くあります。日本人的姓氏种类有近30万种。语法： ～近く
说明： 接尾词，接在数量词后，表示“接近……”、“大约……”。
それでも世界の第2位です。即便如此仍是世界第二。语法： それでも
说明： 接续词，表示转折，“尽管如此”。
第1位は多民族国家のアメリカで，その数は160万を超えています。第一位是多民族国家美国，数量超过160万。语法： ～を超える
说明： 他动词，“超过……”。“を”表示超越的对象。
一方，最も少ないのが韓国で3 0 0程度しかありません。另一方面，最少的是韩国，只有300种左右。1. 一方
说明： 接续词，用于对比，“另一方面”。
2. ～しか～ない
说明： 表示限定，“只有……”，与否定形式连用。
また，世界でいちばん人数の多い姓は「李」で，約1億人いるそうです。此外，世界上人数最多的姓氏是“李”，据说约有1亿人。1. ～で
说明： 中顿形，用于连接句子。
2. ～そうだ
说明： 传闻助动词，表示“听说……”。接在简体形（此处为“いる”）后。
