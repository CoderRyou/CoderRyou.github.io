会话 - 打ち合わせ

（浏览完所有人的方案提要之后）

李： 中井さんの企画は，テレビCMね。「燕醸造のCMを参考にする」と書いてあ るけど，これはどういう意味？

中井： 日本の清涼飲料水のCMなんですが，制作スタッフが中国人なんです。き っと参考になるだろうと思います。

李： ああ，あの評判のCMね。確かに中国人の感性がよく分かるわ。それから， 野田さんの「創作中華料理コンテスト」という企画はどういうもの？

野田： はい。「金星」に合う創作中華料理を募集するんです。コンテストはパ ーティー形式にして，有名な歌手や俳優に審査員として参加してもらいます。

李： コンテストをパーティ一形式に？

野田： はい。選考会の会場は，上海ならノーストンホテル，北京なら華晶酒店 を考えています。 李： ノーストンホテルと華晶酒店？どちらも最高級の場所よ。

野田： ええ。高級ホテルとスター。きっと話題になるだろうと思います。

李： なるほど，それなら，テレビに取り上げてもらうこともできるわね。

（预备会的最后） 李： では，企画会議までに，それぞれの案をできるだけ具体的にまとめてくだ さい。 王： 企画会議は来週の何曜日でしたっけ？

李： 木曜日です。

会话翻译

（浏览完所有人的方案提要之后）

李： 中井的策划是电视广告啊。写着"参考燕酿造的广告"，这是什么意思？

中井： 这是日本清凉饮料的广告，但制作人员是中国人。我觉得一定可以作为参考。

李： 啊，是那个备受好评的广告啊。确实能很好地理解中国人的感受。那么，野田的"创作中华料理比赛"这个策划是怎么回事？

野田： 是的。征集适合"金星"的创作中华料理。比赛以派对形式举行，邀请著名歌手和演员作为评委参加。

李： 比赛以派对形式？

野田： 是的。评选会的会场，上海的话考虑在诺斯顿酒店，北京的话考虑在华晶酒店。

李： 诺斯顿酒店和华晶酒店？这两个都是最高级的场所啊。

野田： 是的。高级酒店和明星。我想一定会成为话题的。

李： 原来如此，那样的话，也可以请电视台来报道呢。

（预备会的最后）

李： 那么，请在策划会议之前，尽可能具体地总结各自的方案。

王： 策划会议是下周的星期几来着？

李： 星期四。

会话语法解析

原文翻译语法解析
李： 中井さんの企画は,テレビCMね。李： 中井的策划是电视广告啊。语法： ～は～ね
说明： 判断句，句末的"ね"表示确认或轻微的感叹。
「燕醸造のCMを参考にする」と書いてあるけど,これはどういう意味?写着"参考燕酿造的广告"，这是什么意思？1. ～を参考にする
说明： 固定表达，"以……为参考"。"参考"是名词，后续"にする"表示"把……作为"。
2. ～と書いてある
说明： "書く"的持续体被动形式（書いてある）表示状态，"写着……"。"と"表示引用的内容。
3. ～けど
说明： 接续助词，口语中用于连接前后句，表示轻微的转折或引出话题。
中井： 日本の清涼飲料水のCMなんですが,制作スタッフが中国人なんです。中井： 这是日本清凉饮料的广告，但制作人员是中国人。1. ～なんですが
说明： "～なのですが"的口语形式，"ですが"表示转折或引出话题，"なん"是"なの"的缩略。
2. ～が～なんです
说明： 强调说明的句型。"が"表示转折，"なんです"是"なのです"的口语形式，用于说明原因或情况。
きっと参考になるだろうと思います。我觉得一定可以作为参考。1. 参考になる
说明： "成为参考"，即"有参考价值"。"に"表示变化的结果。
2. ～だろうと思う
说明： "だろう"表示推测，"と思う"表示说话人的想法。合起来表示"我认为……吧"，是一种委婉的表达方式。
李： ああ，あの評判のCMね。李： 啊，是那个备受好评的广告啊。语法： あの～ね
说明： "あの"指双方都知道的事物，"ね"表示确认。
確かに中国人の感性がよく分かるわ。确实能很好地理解中国人的感受。语法： ～が分かる
说明： "理解……"、"明白……"。"わ"是女性用语，表示轻微的感叹或主张。
それから,野田さんの「創作中華料理コンテスト」という企画はどういうもの?那么，野田的"创作中华料理比赛"这个策划是怎么回事？1. ～という企画
说明： "叫做……的策划"。"という"用于说明内容或名称。
2. どういうもの
说明： "什么样的东西"、"怎么回事"。用于询问事物的性质或内容。
野田： はい。 「金星」に合う創作中華料理を募集するんです。野田： 是的。征集适合"金星"的创作中华料理。1. ～に合う
说明： "适合……"。"に"表示适合的对象。
2. ～を募集する
说明： "募集……"、"征集……"。
3. ～んです
说明： "～のです"的口语形式，用于解释说明或强调理由。
コンテストはパーティー形式にして,有名な歌手や俳優に審査員として参加してもらいます。比赛以派对形式举行，邀请著名歌手和演员作为评委参加。1. ～を～形式にする
说明： "把……做成……形式"。"～にして"是"～にする"的中顿形式。
2. ～に～として参加してもらう
说明： "～に"表示动作的接受者（受邀者），"として"表示身份，"参加してもらう"表示请对方参加。这是一种委婉的请求表达。
李： コンテストをパーティ一形式に?李： 比赛以派对形式？语法： 重复对方的话表示确认，是省略句。完整为"コンテストをパーティー形式にするのですか"。
野田： はい。選考会の会場は,上海ならノーストンホテル,北京なら華晶酒店を考えています。野田： 是的。评选会的会场，上海的话考虑在诺斯顿酒店，北京的话考虑在华晶酒店。1. ～なら
说明： 表示假设条件，"如果……的话"。
2. ～を考えている
说明： "正在考虑……"。"～ている"表示动作的持续状态。
李： ノーストンホテルと華晶酒店?どちらも最高級の場所よ。李： 诺斯顿酒店和华晶酒店？这两个都是最高级的场所啊。1. どちらも
说明： "两者都……"。
2. ～よ
说明： 终助词，女性用语，表示强调或提醒对方注意。
野田： ええ。高級ホテルとスター。きっと話題になるだろうと思います。野田： 是的。高级酒店和明星。我想一定会成为话题的。语法： 話題になる
说明： "成为话题"、"引起热议"。
李： なるほど,それなら,テレビに取り上げてもらうこともできるわね。李： 原来如此，那样的话，也可以请电视台来报道呢。1. なるほど
说明： 副词，表示理解，"原来如此"、"确实"。
2. それなら
说明： 接续词，"那样的话"。
3. ～に取り上げてもらう
说明： "取り上げる"意为"报道、采纳"，"～てもらう"表示请别人为自己做某事。
李： では,企画会議までに，それぞれの案をできるだけ具体的にまとめてください。李： 那么，请在策划会议之前，尽可能具体地总结各自的方案。1. ～までに
说明： 表示时间的截止点，"在……之前"。
2. できるだけ
说明： 副词，"尽可能"、"尽量"。
3. ～をまとめる
说明： "总结……"、"归纳……"。
王： 企画会議は来週の何曜日でしたっけ?王： 策划会议是下周的星期几来着？语法： ～でしたっけ
说明： "～でしたか"的口语形式，"っけ"表示回忆或确认已经知道但暂时忘记的信息。

课文 - 電子メーノレの作法

件名(Subject)：「金星」キャンペーンの企画案

竜虎酒造株式会社「金星」販売促進企画部

佐藤光一様

いつもお世話になっております。JC企画の王です。

先日お話しました「金星」キャンペーンの企画案ができました。

できるだけ早くご説明に伺いたいと思います。

来週のご予定はいかがですか。ご都合のいい時間をお知らせください。

よろしくお願い申し上げます。

株式会社JC企画上海支社

王風 wangf@jckikaku.cn

  メールは手軽な連絡の手段として，世界中で使われている。ただし，仕事のうえでのメールや目上の人に送る場合は，いくつか気をつけなければならないことがある。注意点さえ理解しておけば，メールはとても便利なコミュニケーションの手段だ。

  1. 件名を書く時は，用件を明確にする
    「こんにちは」「はじめまして」などという，メールの内容の分からない表現は避け，件名だけで内容が分かるようにする。

  2. 宛て先と自分の名前を忘れずに入れる
    会社名や部署が分かっている場合は，「〜株式会社〜部」のように宛て先を書き，相手の名前が分かっている場合は，文の始めに「〇〇様」と相手の名前をえれるといい。

  3. 用件は簡潔にする
    要点を整理して，伝えたいことを簡潔に書く。1行は30文字ぐらいで改行し，3〜4行ごとに1行あけると読みやすい。

  4. 最後に署名を入れる
    文の最後には，普通，自分の名前，所属する会社，部署やメールアドレスなどを書いた「署名」を付ける。

课文翻译 - 电子邮件的写法

  邮件作为便捷的联系手段，在全世界被使用。不过，在工作上使用的邮件或给上级发送时，有一些必须注意的事项。只要理解了注意事项，邮件就是非常方便的交流手段。

  1. 写主题时，要明确事项
    避免使用"你好"、"初次见面"等无法了解邮件内容的表达，要让收件人仅通过主题就能明白内容。

  2. 不要忘记写收件人和自己的姓名
    如果知道公司名和部门，就像"～株式会社～部"这样写收件地址，如果知道对方的名字，在文章开头写上"○○様"对方的姓名为好。

  3. 事项要简洁
    整理要点，简洁地写下想传达的内容。每行大约30个字换行，每3～4行空一行的话，会更容易阅读。

  4. 最后加上签名
    在文章的最后，通常要加上写有自己姓名、所属公司、部门或邮件地址等的"签名"。

2. ～で使われている
说明： 被动形式的持续体，"被使用着"。"で"表示范围。

说明： "……的时候"、"……的情况下"。
