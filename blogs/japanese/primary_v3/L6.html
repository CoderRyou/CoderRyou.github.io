会话 - 先輩

中井： 竜虎酒造の佐藤さんなんだけど，なんとぼくの大学の先輩だったんだ。

野田： えつ，先輩？あの佐藤さんが？

中井： そう。バスケットボール部の2年先輩。4年生の時はキャプテンだったんだよ。

野田： ふうん。じゃあ，人気があったんでしょうねえ。

中井： それはもう。まじめだし，みんなから信頼されていたしね。

（王风走进食堂）

中井： あつ，王さん，今からお昼？今日は遅いんだね。

王： 打ち合わせが長引いちやって。何の話してたの？

中井： 竜虎酒造の佐藤さんのことを話していたんだあの人，ぼくの大学の先輩なんだよ。

王： へえ，そうだったの。

中井： そういえば，佐藤さんに李さんのことを聞かれたな。どんな人だって。

野田： それで，中井さんは何て言ったの？

中井： うん。明るくて，仕事熱心な人だって言ったよ。部下の面倒をよく見るし，優しいし，まさに理想的な上司で，...。

野田： そんなことを聞くということは，佐藤さん，李さんのことが気になるのかしら。

王： うん，そうかもね。

会话翻译

中井： 关于龙虎酒业的佐藤先生，他居然是我的大学学长。

野田： 诶，学长？那位佐藤先生吗？

中井： 是的。篮球部的学长，比我高两届。大四的时候还是队长呢。

野田： 哦～那当时很受欢迎吧？

中井： 那当然了。又认真，又受大家信赖。

（王风走进食堂）

中井： 啊，王先生，现在才吃午饭？今天挺晚的啊。

王： 会议拖得太久了。你们在聊什么？

中井： 在聊龙虎酒业的佐藤先生，他是我大学学长。

王： 诶，是这样啊。

中井： 说起来，佐藤先生还问起李小姐的事了呢。问我她是个什么样的人。

野田： 那，中井你怎么回答的？

中井： 嗯。我说她是个开朗、工作热心的人。很照顾下属，又很温柔，简直是理想的上司……

野田： 会这么打听，是不是佐藤先生对李小姐有点在意啊？

王： 嗯，可能吧。

会话语法解析

原文翻译语法解析
中井： 竜虎酒造の佐藤さんなんだけど，なんとぼくの大学の先輩だったんだ。中井： 关于龙虎酒业的佐藤先生，他居然是我的大学学长。1. ～なんだけど
说明： 口语中“～なのだが”的省略形式，用于引出话题，相当于“关于……”。
2. なんと
说明： 感叹词，表示惊讶、赞叹，相当于“居然”、“竟然”。
3. ～だったんだ
说明： “～だったのだ”的口语形式，表示强调说明，说明一个事实。
野田： えつ，先輩？あの佐藤さんが？野田： 诶，学长？那位佐藤先生吗？语法： あの～が？
说明： 这里用“が”表示疑问，强调主语，带有惊讶的语气。
中井： そう。バスケットボール部の2年先輩。4年生の時はキャプテンだったんだよ。中井： 是的。篮球部的学长，比我高两届。大四的时候还是队长呢。1. ～の時は
说明： “……的时候”。
2. ～だったんだよ
说明： 同第一句，强调说明，句末的“よ”用于提醒对方注意。
野田： ふうん。じゃあ，人気があったんでしょうねえ。野田： 哦～那当时很受欢迎吧？语法： ～んでしょうねえ
说明： “～のでしょうね”的口语形式，表示推测，并带有确认的语气。“ねえ”是“ね”的拖长音，表示感叹。
中井： それはもう。まじめだし，みんなから信頼されていたしね。中井： 那当然了。又认真，又受大家信赖。1. それはもう
说明： 固定表达，表示“那当然”、“毫无疑问”。
2. ～だし～し
说明： 列举原因或特征，“既……又……”。
3. ～から信頼されていた
说明： “から”表示动作的发出者，“信頼されていた”是“信頼する”的被动态过去进行时，表示“过去一直受到信赖”。
中井： あつ，王さん，今からお昼？今日は遅いんだね。中井： 啊，王先生，现在才吃午饭？今天挺晚的啊。1. 今からお昼？
说明： 省略句，完整为“今からお昼を食べるのですか”。
2. ～んだね
说明： “～のだね”的口语形式，表示确认或理解，带有感叹语气。
王： 打ち合わせが長引いちやって。何の話してたの？王： 会议拖得太久了。你们在聊什么？1. ～が長引く
说明： 表示“某事拖延”。
2. ～ちやって
说明： 口语中“～てしまって”的约音形式，表示动作的完成或遗憾的语气。
3. 何の話してたの？
说明： 口语省略，“何の話をしていたの？”询问对方在聊什么。
中井： 竜虎酒造の佐藤さんのことを話していたんだあの人，ぼくの大学の先輩なんだよ。中井： 在聊龙虎酒业的佐藤先生，他是我大学学长。1. ～ことを話していた
说明： “話す”的过去进行时，表示“一直在谈论……”。
2. ～なんだよ
说明： 强调说明，解释原因或情况。
王： へえ，そうだったの。王： 诶，是这样啊。语法： そうだったの
说明： 表示得知新信息时的回应，“原来是这样啊”。
中井： そういえば，佐藤さんに李さんのことを聞かれたな。どんな人だって。中井： 说起来，佐藤先生还问起李小姐的事了呢。问我她是个什么样的人。1. そういえば
说明： 转换话题或突然想起某事，“说起来”。
2. ～に聞かれた
说明： “聞く”的被动形式，表示“被……问”。
3. ～な
说明： 终助词，表示确认或回忆。
4. どんな人だって
说明： 间接引语，“って”表示引用，相当于“と”，这里省略了“聞かれた”的内容。
野田： それで，中井さんは何て言ったの？野田： 那，中井你怎么回答的？1. それで
说明： 接续词，表示“于是”、“那么”。
2. 何て言ったの？
说明： “何と言ったの？”的口语形式，询问说了什么。
中井： うん。明るくて，仕事熱心な人だって言ったよ。部下の面倒をよく見るし，優しいし，まさに理想的な上司で，...。中井： 嗯。我说她是个开朗、工作热心的人。很照顾下属，又很温柔，简直是理想的上司……1. 明るくて
说明： 形容词て形，连接两个描述。
2. ～だって言った
说明： “～だと言った”的口语形式，表示“说了……”。
3. ～をよく見る
说明： 固定搭配，“よく”表示经常，“面倒を見る”意为“照顾”。
4. ～し～し
说明： 列举理由或特征。
5. まさに～で
说明： “まさに”表示“正是”，“で”是“だ”的中顿形。
野田： そんなことを聞くということは，佐藤さん，李さんのことが気になるのかしら。野田： 会这么打听，是不是佐藤先生对李小姐有点在意啊？1. ～ということは
说明： 表示“也就是说……”，用于根据前文进行推测。
2. ～が気になる
说明： 固定表达，“在意……”。
3. ～のかしら
说明： 女性用语，表示不确定的推测或疑问。
王： うん，そうかもね。王： 嗯，可能吧。语法： そうかもね
说明： “そうかもしれないね”的省略形式，表示不确定的赞同。

课文 - 「はしの文化」さまざま

  手，はし，フォーク，ナイフ，スプーンなど，食べる時に何を使うかは，食事の內容や習慣によって違う。

  日本では，洋食はナイフとフォーク，スプーンで食べ，和食は，はしを使う。中国や韓国では，スープをスプーンやれんげを使って食べるが，日本のみそ汁は，おわんを持ち上げ，直接口を つけて食べる。具を食べる時はもちろんはしを使う。

  「はし」といっても，その形や材料は国や地域によって違う。中国のはしは長くて，先端が丸く太さがあまり変わらない。材料は本や竹，プラスチックのほか，玉や金属のものもある。韓国 のはしは中国より少し短く，やや平らな形をしている。ステンレスなど，金属製のものが一般的だ。日本のはしは韓国のものに比べてさらに短く，先のほうは細くなっていて，とがってい る。材質は木や竹が多く，漆が塗られていることもある。

  日本の家庭では，自分専用のはしや茶わんを使う。食事は1人1人，お皿や茶わんに分けて出されるのが一般的だが，大皿から料理を取る時は，取りばしを使うのが，正式なマナーだとされ る。

  食事の時にはしを使うのは，中国，韓国，ベトナム，日本などで，世界の人口の約3割だというが，同じ「はしを使う」文化も，実にさまざまだ。

课文翻译 - 筷子的各种各样的文化

  手、筷子、叉子、刀子、勺子等，吃饭时使用什么工具，因饮食内容和习惯的不同而有所差异。

  在日本，西餐用刀、叉和勺子吃，日餐则用筷子。在中国和韩国，喝汤时会用勺子或汤勺，而日本的味噌汤则是把碗端起来直接喝。当然，吃里面的菜时要用筷子。

  虽说都叫“筷子”，其形状和材料也因国家和地区而异。中国的筷子长，前端圆，粗细变化不大。材料除了木头、竹子、塑料，还有玉或金属的。韩国的筷子比中国的稍短，形状略扁。一般是不锈钢等金属制品。日本的筷子比韩国的更短，前端细而尖。材质多是木头或竹子，有时会涂上漆。

  在日本家庭里，人们会使用自己专用的筷子和饭碗。一般吃饭时，会给每个人分好碟子和碗，但从大盘里取菜时，使用公筷被认为是正式的礼节。

  据统计，吃饭时使用筷子的有中国、韩国、越南、日本等，约占世界人口的三成，但同样是用筷子的文化，实际上也千差万别。

说明： 疑问词“何を使うか”作为主语，用“は”提示主题。
2. ～によって違う
说明： 表示“根据……不同”。

说明： 表示范围，“在……（国家）”。
2. ～で食べ
说明： “で”表示工具，“食べ”是“食べる”的中顿形。

说明： “持ち上げる”的中顿形，意为“端起”。
