会话 - 東京本社

（山田接到人事部年轻职员打来的电话）

社員： もしもし，山田さん。うちの青木主任が，ちょっとお話をしたいと言っていました。

山田： わたしにですか。

社員： ええ。青木主任は，午後なら何時でもかまわないそうです。山田さんのご都合はいかがですか。

山田： そうですね...。1時半はどうでしょう。

社員： ありがとうございます。そのようにお伝えします。

（下午，山田来到人事部见青木）

青木： お忙しいところ，すみません。実は，上海支社で進めている「金星」プロジェクトが，思ったより大変な仕事になりそうなんです。

山由： そうですか。

青木： それで，東京本社でも「金星」プロジェクトの担当者を決めることになりました。突然ですが，山田さん，この仕事をしてもらえませんか。

山田： わたしが「金星」プロジェクトの担当者に?

青木： はい。上海支社の李さんからの要望なんです。

山田： 李さんが，わたしを推薦してくれたんですか。

青木： ええ。山田さんは上海支社にいたこともあるし，中国の事情に詳しいからということでした。

山田： 分かりました。中国と関係のある仕事なら，ぜひやってみたいと思います。

会话翻译

（山田接到人事部年轻职员打来的电话）

员工： 喂，山田先生。我们部门的青木主任说想和您谈谈。

山田： 是找我吗？

员工： 是的。青木主任说如果是下午的话，什么时间都可以。您的时间方便吗？

山田： 嗯……一点半怎么样？

员工： 谢谢。我会那样转告的。

（下午，山田来到人事部见青木）

青木： 在您百忙之中打扰，不好意思。其实，上海分公司正在推进的“金星”项目，看起来会比预想的要棘手。

山田： 这样啊。

青木： 因此，东京总公司也决定要确定“金星”项目的负责人了。虽然很突然，但山田先生，能请您负责这项工作吗？

山田： 让我当“金星”项目的负责人？

青木： 是的。这是上海分公司李小姐提出的要求。

山田： 是李小姐推荐我的吗？

青木： 是的。说是（因为）山田先生您曾在上海分公司工作过，而且对中国的情况很熟悉。

山田： 我明白了。如果是与中国相关的工作，我很想尝试一下。

会话语法讲解

原文翻译语法解析
社員： もしもし，山田さん。うちの青木主任が，ちょっとお話をしたいと言っていました。员工： 喂，山田先生。我们部门的青木主任说想和您谈谈。1. ～が～と言っている
说明： 表示第三人称的说话内容。“青木主任”是主语，用“が”表示；引用内容后接“と言っている”表示“说……”。
2. お話をしたい
说明： “話す”的敬语形式，用“お～する”的自谦结构表达“想和您谈话”的意愿。
山田： わたしにですか。山田： 是找我吗？语法： ～に + ですか
说明： “に”表示动作的对象，“ですか”读升调表示确认。
社員： ええ。青木主任は，午後なら何時でもかまわないそうです。员工： 是的。青木主任说如果是下午的话，什么时间都可以。1. ～なら
说明： 表示假设或限定，“如果是……的话”。
2. 何時でもかまわない
说明： “何時でも”表示“任何时候”；“かまわない”是“構う”的否定形，意为“没关系、不介意”。
3. ～そうです
说明： 传闻助动词，接在动词基本形后，表示“听说……”。
山田さんのご都合はいかがですか。您的时间方便吗？语法： ご～はいかがですか
说明： 礼貌询问的表达。“ご都合”是“都合”的敬语形式，“いかがですか”是“どうですか”的礼貌说法。
山田： そうですね...。1時半はどうでしょう。山田： 嗯……一点半怎么样？语法： ～はどうでしょう
说明： 提出建议的委婉表达，相当于“……怎么样？”。
社員： ありがとうございます。そのようにお伝えします。员工： 谢谢。我会那样转告的。语法： お～する
说明： 自谦语结构，“お伝えします”是“伝えます”的自谦说法，表示“（我）转告”。
青木： お忙しいところ，すみません。青木： 在您百忙之中打扰，不好意思。语法： お～ところ
说明： 固定寒暄表达，“お忙しいところ”意为“在您正忙的时候”，用于开启对话的礼貌客套。
実は，上海支社で進めている「金星」プロジェクトが，思ったより大変な仕事になりそうなんです。其实，上海分公司正在推进的“金星”项目，看起来会比预想的要棘手。1. ～で進めている
说明： “で”表示动作发生的场所；“進めている”是“進める”的持续体，意为“正在推进”。
2. 思ったより
说明： 固定表达，“比预想的……”。“思った”是“思う”的过去式作定语。
3. ～そうなんです
说明： 样态助动词“そうだ”的礼貌强调形式，接在动词连用形/形容词词干后，表示“看起来……”。此处“なりそう”是“なる”的样态。
青木： それで，東京本社でも「金星」プロジェクトの担当者を決めることになりました。青木： 因此，东京总公司也决定要确定“金星”项目的负责人了。语法： ～ことになりました
说明： 表示客观决定或结果，常用于机构、组织做出的决定，意为“决定……”。
突然ですが，山田さん，この仕事をしてもらえませんか。虽然很突然，但山田先生，能请您负责这项工作吗？1. ～ですが
说明： 表示转折，“虽然……但是”。
2. ～てもらえませんか
说明： “～てもらう”的否定疑问形式，表示委婉的请求，意为“能请您……吗？”。
山田： わたしが「金星」プロジェクトの担当者に?山田： 让我当“金星”项目的负责人？语法： ～が～に
说明： “が”提示主语，“に”表示角色或身份，此处省略了谓语（なる等），以疑问语气表示确认。
青木： はい。上海支社の李さんからの要望なんです。青木： 是的。这是上海分公司李小姐提出的要求。语法： ～からの要望
说明： “から”表示来源，“の”连接名词，“要望”意为“要求、请求”。“なんです”是“のです”的口语形式，用于解释说明。
山田： 李さんが，わたしを推薦してくれたんですか。山田： 是李小姐推荐我的吗？语法： ～てくれた + んですか
说明： “～てくれる”表示他人为自己做某事，“くれた”是其过去式；“んですか”表示确认或寻求解释。
青木： ええ。山田さんは上海支社にいたこともあるし，中国の事情に詳しいからということでした。青木： 是的。说是（因为）山田先生您曾在上海分公司工作过，而且对中国的情况很熟悉。1. ～こともある
说明： “ことも”是“こと”的强调形式，“ある”表示“有（过）”。整体表示“曾经有过……的经历”。
2. ～し
说明： 接续助词，列举理由或事实，“既……又……”。
3. ～に詳しい
说明： 固定搭配，“对……熟悉”。“に”表示对象。
4. ～からということでした
说明： “から”表示原因；“ということでした”是间接引用的表达，意为“说是……”。
山田： 分かりました。中国と関係のある仕事なら，ぜひやってみたいと思います。山田： 我明白了。如果是与中国相关的工作，我很想尝试一下。1. ～と関係がある
说明： “和……有关”。“と”表示关联的对象。
2. ～なら
说明： 表示条件，“如果……的话”。
3. ～てみたい
说明： “～てみる”表示尝试，“～たい”表示愿望，合起来意为“想尝试……”。
4. ～と思います
说明： 表示说话人的想法，“我认为……”。

课文 - 近年サラリーマン事情

  かつて，日本のサラリーマンといえば，朝早く家を出て家に帰るのは深夜になり，残業や休日出勤をするのは当たり前でした。近年，週休2日制の導入や労働時間を減らす取り組みなどが実施され，サラリーマンの生活は変化してきました。

  週休2日制は1980年代から企業に導入されてきました。1992年には国家公務員に，2002年には小•中学校にこの制度が導入されました。2005年の調査によると，日本の企業の89%が，週休2日制を実施しています。

  さらに，フレックスタイム制度を導入する会社も増えています。これは，自分の出勤と退勤の時間を自由に決められる制度です。労働者が自分の生活と仕事のバランスを取りながら，働くことができるように設けられました。

  また，日本の会社には，育児休暇の制度がありますが，これまでこの制度を利用するのは，ほとんどが女性でした。男性の場合，会社での理解が得られないのではないか，出世の妨げとなるのではないかと考える人が多かったからです。近年，家族との時間を大切にしたいと考えて，この制度を利用する男性が見られるようになりました。

  このように，日本のサラリーマンの働き方は，かつてと比べると大きく変化しています。都会での時間に追われる生活をやめて，田舎に引っ越す人もいます。

课文翻译

  曾经，说起日本的上班族，早上早早出门，深夜才回家，加班和休息日出勤是理所当然的。近年来，随着每周双休制的引入以及减少劳动时间等措施的实施，上班族的生活已经发生了变化。

  每周双休制从20世纪80年代起就开始被引入企业。1992年，这项制度被引入国家公务员系统；2002年，又被引入中小学。根据2005年的调查，日本89%的企业实行了每周双休制。

  此外，引入弹性工作制的公司也在增加。这是一种可以自由决定自己上下班时间的制度。为了让劳动者能够在兼顾生活与工作平衡的同时工作而设立了这一制度。

  另外，日本公司虽然有育儿假制度，但迄今为止利用这一制度的大多是女性。这是因为很多男性认为，在公司可能得不到理解，或者这可能成为升职的障碍。近年来，由于想要珍惜与家人相处的时间，可以看到开始有男性利用这一制度了。

  这样，日本上班族的工作方式，与过去相比发生了巨大的变化。也有人放弃在大都市被时间追赶的生活，搬到乡下。

课文语法讲解

原文翻译语法解析
かつて，日本のサラリーマンといえば，朝早く家を出て家に帰るのは深夜になり，残業や休日出勤をするのは当たり前でした。曾经，说起日本的上班族，早上早早出门，深夜才回家，加班和休息日出勤是理所当然的。1. ～といえば
说明： 表示提起某个话题，“说到……”。
2. ～のは～になる
说明： “～のは”将前半句名词化作主语；“になる”表示变化的结果。
3. ～のは当たり前でした
说明： “～のは”名词化主语部分；“当たり前”意为“理所当然”，整体表示“……是理所当然的”。
近年，週休2日制の導入や労働時間を減らす取り組みなどが実施され，サラリーマンの生活は変化してきました。近年来，随着每周双休制的引入以及减少劳动时间等措施的实施，上班族的生活已经发生了变化。1. ～や～など
说明： “や”表示列举，“など”表示“等等”，合起来列举事物。
2. ～が実施される
说明： 被动形式，表示“被实施”。
3. ～てきました
说明： “～てくる”的过去式，表示变化从过去持续到现在，“……起来了”。
週休2日制は1980年代から企業に導入されてきました。每周双休制从20世纪80年代起就开始被引入企业。语法： ～から～てきました
说明： “から”表示起点；“導入されてきました”是“導入される”的て形+くる，表示从过去持续到现在的动作。
1992年には国家公務員に，2002年には小•中学校にこの制度が導入されました。1992年，这项制度被引入国家公务员系统；2002年，又被引入中小学。语法： ～に導入される
说明： 被动形式，“被引入到……”。“に”表示对象。
2005年の調査によると，日本の企業の89%が，週休2日制を実施しています。根据2005年的调查，日本89%的企业实行了每周双休制。1. ～によると
说明： 表示依据，“根据……”。
2. ～を実施している
说明： “实施……”的持续体，表示状态的持续。
さらに，フレックスタイム制度を導入する会社も増えています。此外，引入弹性工作制的公司也在增加。语法： ～も増えている
说明： “も”表示“也”；“増えている”是“増える”的持续体，表示持续增加的状态。
これは，自分の出勤と退勤の時間を自由に決められる制度です。这是一种可以自由决定自己上下班时间的制度。语法： ～られる（可能形）
说明： “決められる”是“決める”的可能形，表示“能够决定”。
労働者が自分の生活と仕事のバランスを取りながら，働くことができるように設けられました。这是为了让劳动者能够在兼顾生活与工作平衡的同时工作而设立的。1. ～ながら
说明： 表示动作同时进行，“一边……一边……”。
2. ～ことができる
说明： 表示可能，“能够……”。
3. ～ように設けられる
说明： “ように”表示目的；“設けられる”是“設ける”的被动形，“被设立”。
また，日本の会社には，育児休暇の制度がありますが，これまでこの制度を利用するのは，ほとんどが女性でした。此外，日本公司虽然有育儿假制度，但迄今为止利用这一制度的大多是女性。1. ～には～がある
说明： “には”提示地点或范围，“有……”。
2. ～のは～が～でした
说明： “～のは”名词化主语（利用制度这件事），“が”提示主语（女性），“でした”是判断助动词的过去式。
男性の場合，会社での理解が得られないのではないか，出世の妨げとなるのではないかと考える人が多かったからです。这是因为很多男性认为，在公司可能得不到理解，或者这可能成为升职的障碍。1. ～場合
说明： 表示情况，“……的情况下”。
2. ～のではないか
说明： 表示推测或不确切的判断，“是不是……呢？”、“恐怕……吧”。
3. ～からです
说明： 用于说明原因，“是因为……”。

说明： “大切にする”意为“珍惜”，“～たい”表示愿望。
2. ～が見られるようになる
说明： “見られる”是“見る”的可能形，“能够看到”；“ようになる”表示变化，“变得……”。
このように，日本のサラリーマンの働き方は，かつてと比べると大きく変化しています。这样，日本上班族的工作方式，与过去相比发生了巨大的变化。1. このように
说明： 总结性表达，“像这样”。
2. ～と比べると
说明： “和……相比”。“と”表示比较的对象；“比べると”是“比べる”的假定形，表示“如果比较的话”。
3. ～ている
说明： 表示变化结果的持续状态。

说明： 被动形式，“被……追赶”，引申为“被……驱使”。
2. ～をやめて
说明： “やめる”的中顿形，表示“放弃……”。
