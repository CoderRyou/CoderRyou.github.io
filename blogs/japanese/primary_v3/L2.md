## 会话 - あいさつ

（大山走进会议室）
天山：すみません，お待たせしました。
佐藤：どうもごぶさたしています。
大山：本当にお久しぶりです。佐藤さんといっしょに
仕事をするのは，4年ぶりですね。
佐藤：ええ。早いものですね。その節はお世話になりました。
大山：いいえ，こちらこそ。
佐藤：今回もよろしくお願いします。
（大山的同事来到会议室）
同僚：お話中，失礼します。JC企画の李さんがいらっしゃいました。
大山：あ，そうですか。じゃあ，こちらにお願いします。
同僚：はい。
（李秀丽走进会议室）
李：こんにちは。
大山：こんにちは，李さん。紹介します。こちらは，今度の「金星」プロジェクトの責任者で，佐藤です。
李、佐藤：はじめまして...。（说着互相打量）あれつ，昨日の...！

| 原文 | 中文翻译 | 语法解析 |
| ------ | --------- | ---------- |
| **天山**：すみません、お待たせしました。 | **天山**：对不起，让您久等了。 | **お待たせしました**：是「待たせる」（让…等待）的自谦语形式。「お＋动词ます形去ます＋する」构成自谦表达，表示自己的动作给对方带来影响。 |
| **佐藤**：どうもごぶさたしています。 | **佐藤**：好久不见了。 | **ごぶさたしています**：「ごぶさた」是「無沙汰」的礼貌形式，意为“久未问候”。「しています」表示状态持续，是固定寒暄语。 |
| **大山**：本当にお久しぶりです。佐藤さんといっしょに仕事をするのは、4年ぶりですね。 | **大山**：真的是好久不见。和佐藤先生一起工作，已经是时隔4年了呢。 | **お久しぶりです**：固定寒暄语，“好久不见”。「お＋形容词/形容动词」构成礼貌表达。<br>**～のは…ぶりですね**：「の」名词化从句，「は」提示主题。「～ぶり」表示“时隔…之后又…”。 |
| **佐藤**：ええ。早いものですね。その節はお世話になりました。 | **佐藤**：是啊。时间过得真快啊。那时承蒙您关照了。 | **早いものですね**：「もの」在此表示感叹，带有“真是…”的语气。<br>**その節**：指“当时、那时”。<br>**お世話になりました**：固定表达，意为“承蒙关照了”。 |
| **大山**：いいえ、こちらこそ。 | **大山**：不，我才是（承蒙您关照了）。 | **こちらこそ**：意为“我才是、彼此彼此”，用于谦逊回应对方的感谢或道歉。 |
| **佐藤**：今回もよろしくお願いします。 | **佐藤**：这次也请多多关照。 | **よろしくお願いします**：固定表达，用于请求对方关照或协助。 |
| **同僚**：お話中、失礼します。JC企画の李さんがいらっしゃいました。 | **同事**：打扰一下你们的谈话。JC策划公司的李先生到了。 | **お話中**：「お＋名词＋中」表示“正在…中”。<br>**失礼します**：固定表达，意为“失礼了、打扰一下”。<br>**いらっしゃいました**：是「来る」的尊敬语，表示“（尊贵的客人）来了”。 |
| **大山**：あ、そうですか。じゃあ、こちらにお願いします。 | **大山**：啊，是吗。那么，请带到这边来。 | **こちらに**：「こちら」是礼貌的方位词，“这边”。<br>**お願いします**：此处表示请求对方做某事，可译为“拜托了、麻烦了”。 |
| **同僚**：はい。 | **同事**：好的。 | 应答词。 |
| **李**：こんにちは。 | **李**：你们好。 | 白天使用的问候语。 |
| **大山**：こんにちは、李さん。紹介します。こちらは、今度の「金星」プロジェクトの責任者で、佐藤です。 | **大山**：你好，李小姐。我来介绍一下。这位是这次“金星”项目的负责人，佐藤先生。 | **紹介します**：自我介绍或介绍他人时的固定开场白。<br>**こちらは…で、…です**：「こちら」礼貌地指代“这位”。「で」是「です」的中顿形，连接两个描述。 |
| **李、佐藤**：はじめまして…。（说着互相打量）あれっ、昨日の…！ | **李、佐藤**：初次见面…。（互相打量）哎呀，你是昨天的…！ | **はじめまして**：初次见面时的固定表达。<br>**あれっ**：感叹词，表示惊讶或突然想起某事。<br>**昨日の**：后面省略了「人」，意为“昨天的那个人”。 |

## 课文 - あいさつの時の礼儀

&emsp;&emsp;日本人同士があいさつする時は，普通お辞儀をします。中国ではあいさつをする時，握手をするのが普通ですが，日本人は，握手よりもお辞儀をするのが一般的です。お辞儀をする時は腰から上を前のほうに傾けます。お辞儀の角度には，首を曲げるだけの軽いものから深く頭を下げるものまで，いろいろあります。普通，初対面の時には深く頭を下げて，丁寧なお辞儀をします。
&emsp;&emsp;仕事で初対面の人に会った時は，名刺を出します。名刺の出し方と受け取り方にも決まりがあります。相手が目上の時は，まず自分から先に名刺を出して深いお辞儀をし，次に相手の名刺をお辞儀をしながら受け取ります。出す時も受け取る時も，両手を使うほうが丁寧です。
&emsp;&emsp;また，話す內容にも気をつけなければなりません。初対面の時は，個人的なことを聞かないのが普通です。特に仕事の場で出会った相手には，個人的な質問は避けたほうがいいでしょう。結婚しているかどうか，給料はいくらか，年はいくつかなどは，普通は聞きません。
&emsp;&emsp;大事なのは，相手に不快感を与えないことです。いくら丁寧な言葉であいさつしても，怖い顔をしていては，相手にいい印象を与えることはできません。気持ちがこもっているかどうかは，予想以上に相手に伝わってしまうものです。

| 原文 | 中文翻译 | 语法解析 |
| ------ | --------- | ---------- |
| 日本人同士があいさつする時は、普通お辞儀をします。 | 日本人之间互相问候时，通常鞠躬。 | **同士**：接尾词，表示“彼此之间、互相”。<br>**お辞儀をします**：「お辞儀」是名词，加「する」构成动词，意为“鞠躬”。 |
| 中国ではあいさつをする時、握手をするのが普通ですが、日本人は、握手よりもお辞儀をするのが一般的です。 | 在中国问候时，通常握手，但在日本，比起握手，鞠躬更为普遍。 | **では**：表示范围，“在…（国家/地区）”。<br>**のが普通ですが**：「の」名词化从句，「が」表示转折。<br>**よりも**：表示比较，“比起…”。<br>**のが一般的です**：同上，「の」名词化，作主语。 |
| お辞儀をする時は腰から上を前のほうに傾けます。 | 鞠躬时，将腰部以上向前倾。 | **腰から上**：「から」表示起点，“从腰部往上”。<br>**を傾けます**：「を」表示动作对象，「傾ける」是他动词，“使…倾斜”。 |
| お辞儀の角度には、首を曲げるだけの軽いものから深く頭を下げるものまで、いろいろあります。 | 鞠躬的角度有多种，从只弯一下脖子的轻微鞠躬，到深深低头的鞠躬都有。 | **には**：「に」表示存在范围，「は」提示主题。<br>**～から～まで**：表示范围“从…到…”。<br>**もの**：此处指代“种类、类型”。<br>**いろいろあります**：表示“有各种各样”。 |
| 普通、初対面の時には深く頭を下げて、丁寧なお辞儀をします。 | 通常，初次见面时会深深低头，郑重地鞠躬。 | **初対面の時には**：「には」强调时间场合。<br>**深く頭を下げて**：「て形」连接两个动作，表示方式或先后。<br>**丁寧な**：形容动词连体形，修饰名词。 |
| 仕事で初対面の人に会った時は、名刺を出します。 | 在工作中初次与人见面时，要交换名片。 | **で**：表示场合或原因，“因工作…”。<br>**に会った時**：「に」表示动作对象，「た時」表示“…的时候”。 |
| 名刺の出し方と受け取り方にも決まりがあります。 | 名片的递出方式和接收方式也有规定。 | **にも**：「に」表示对象，「も」表示“也”。<br>**決まりがあります**：意为“有规定、有规矩”。 |
| 相手が目上の時は、まず自分から先に名刺を出して深いお辞儀をし、次に相手の名刺をお辞儀をしながら受け取ります。 | 当对方是上级时，首先要自己先递出名片并深鞠躬，然后一边鞠躬一边接过对方的名片。 | **目上**：名词，“上级、长辈”。<br>**まず…次に**：表示顺序“首先…接着…”。<br>**～ながら**：表示两个动作同时进行，“一边…一边…”。 |
| 出す時も受け取る時も、両手を使うほうが丁寧です。 | 无论是递出还是接收时，用双手都更显郑重。 | **～も～も**：表示“两者都…”。<br>**ほうが丁寧です**：「ほうが」表示比较，“…一方更…”。 |
| また、話す内容にも気をつけなければなりません。 | 此外，也必须注意谈话的内容。 | **～に気をつける**：固定搭配，“注意…、留心…”。<br>**なければなりません**：表示义务，“必须…”。 |
| 初対面の時は、個人的なことを聞かないのが普通です。 | 初次见面时，通常不问个人事情。 | **個人的なこと**：个人隐私、私事。<br>**聞かないのが普通です**：「の」名词化，作主语。 |
| 特に仕事の場で出会った相手には、個人的な質問は避けたほうがいいでしょう。 | 尤其是在工作场合遇到的人，最好避免询问个人问题。 | **～には**：表示对象，“对…来说”。<br>**避けたほうがいい**：固定表达，“最好避免…”。<br>**でしょう**：表示委婉断定或建议。 |
| 結婚しているかどうか、給料はいくらか、年はいくつかなどは、普通は聞きません。 | 是否已婚、工资多少、年龄多大等等，通常是不问的。 | **～かどうか**：表示“是否…”。<br>**いくらか**：「か」表示不确定，“多少（钱）”。<br>**いくつか**：“多大（年龄）”。<br>**などは**：「など」列举，“等等”；「は」提示主题并带有对比含义。 |
| 大事なのは、相手に不快感を与えないことです。 | 重要的是，不要让对方感到不快。 | **大事なのは…ことです**：「の」与「こと」都是形式名词，此处「のは」提示主语从句，「ことです」作谓语，说明内容是“（做到）…这件事”。 |
| いくら丁寧な言葉であいさつしても、怖い顔をしていては、相手にいい印象を与えることはできません。 | 无论用多么礼貌的语言问候，如果表情严肃，也无法给对方留下好印象。 | **いくら～ても**：表示“无论怎么…也”。<br>**～ていては**：「ては」表示假设条件，后项多为否定或消极结果，“如果…的话，就（不）…”。 |
| 気持ちがこもっているかどうかは、予想以上に相手に伝わってしまうものです。 | 是否饱含诚意，会超乎想象地传递给对方。 | **～かどうかは**：同上，表示“是否…”。<br>**予想以上に**：副词，“超乎想象地”。<br>**～てしまう**：此处表示“完全地、彻底地”。<br>**ものです**：表示本质或常理，“本来就是…的东西”，强调这是一种普遍事实。 |

## 📚 本课核心语法点总结

| 语法点 | 说明 | 例句 |
| -------- | ------ | ------ |
| **お・ご～する** | 自谦语表达，通过降低自己来表示对对方的尊敬。 | **お待たせしました**（让您久等了） |
| **～ぶり** | 表示“时隔…之后又…”。 | **4年ぶりですね**（时隔4年了呢） |
| **～ものだ／ものですね** | 表示感叹、本质或常理。 | **早いものですね**（时间过得真快啊） |
| **～かどうか** | 表示“是否…”。 | **結婚しているかどうか**（是否已婚） |
| **いくら～ても** | 表示“无论怎么…也”。 | **いくら丁寧な言葉でも**（无论多么礼貌的语言） |
| **～ほうがいい／～ほうが丁寧です** | 表示比较，“…更好/更郑重”。 | **両手を使うほうが丁寧です**（用双手更郑重） |
| **～なければなりません** | 表示义务，“必须…”。 | **気をつけなければなりません**（必须注意） |
| **～ものだ**（表示本质） | 强调事物的本质、常理或普遍性。 | **伝わってしまうものです**（本来就是会传递的） |

> 注：本课重点在于**敬语的使用**（自谦语「お待たせしました」、尊敬语「いらっしゃいました」）以及**日本商务礼仪与文化**（鞠躬、名片交换、谈话禁忌）。理解这些表达有助于在实际交流中更得体地应对正式场合。
