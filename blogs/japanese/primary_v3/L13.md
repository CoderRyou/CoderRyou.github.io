## 会话 - スピーチの依頼

（工作休息时间，山田来到吉田处）

**山田：** 課長，お忙しいところ申し訳ありませんが，ちょっとよろしいですか。

**吉田：** うん，何だい？

**山田：** 実は，折り入ってお願いしたいことがあるんですが…。

**吉田：** どうしたの？改まって。

**山田：** ええ。わたしの披露宴のことなんです。課長にスピーチをお願いしたいと思いまして…。

**吉田：** ぼくがスピーチを？

**山田：** はい。ぜひお願いいたします。

**吉田：** ううん，あまり自信がないけどなあ。しかし，君の頼みだ。引き受けるか。

**山田：** ありがとうございます。

（转入婚宴客人的话题）

**吉田：**　もしかして，上海の李さんも来るの？

**山田：**　いいえ。今回のプロジェクトが忙しくて，どうしても中国を離れられないそうです。

**吉田：**　そうか，残念だね。ぼくも，李さんには会いたかったんだけど。

**山田：**　ええ。でも，ちょうどその時期に上海支社の王さんが取材で日本に来ているので，王さんが出席してくれることになりました。

**吉田：**　それはよかった。何といっても，上海は，君と町子さんが出会った思い出の場所だからね。

### 会话翻译 - 演讲邀请

（工作休息时间，山田来到吉田处）  

**山田：** 科长，打扰您百忙之中，请问现在方便吗？  

**吉田：** 嗯，什么事？  

**山田：** 其实，我有一件私事想恳求您……  

**吉田：** 怎么了？这么正式。  

**山田：** 嗯。是关于我的婚宴的事。我想请科长您来致辞……  

**吉田：** 我来致辞？  

**山田：** 是的。拜托您了。  

**吉田：** 嗯……我没什么自信啊。不过，既然是你的请求，那我就答应吧。  

**山田：** 谢谢您。  

**吉田：** 说起来，上海的李先生/小姐也来吗？  

**山田：** 不。听说因为这次的项目很忙，怎么也离不开中国。  

**吉田：** 是吗，真遗憾啊。我也很想见李先生的。  

**山田：** 嗯。不过，正好那个时候上海分公司的王先生要来日本采访，所以王先生会出席。  

**吉田：** 那太好了。毕竟，上海是你和町子女士相识的、充满回忆的地方嘛。

| いいえ。今回のプロジェクトが忙しくて，どうしても中国を離れられないそうです。 | 不。听说因为这次的项目很忙，怎么也离不开中国。 | 「が忙しくて」：原因；「どうしても〜ない」：无论如何也不；「離れられない」：可能动词否定形；「そうです」：传闻 |
| そうか，残念だね。ぼくも，李さんには会いたかったんだけど。 | 是吗，真遗憾啊。我也很想见李先生的。 | 「そうか」：男性口语；「だね」：简体+终助词；「には」：对比强调对象；「会いたかったんだけど」：过去愿望+说明背景+委婉转折 |
| ええ。でも，ちょうどその時期に上海支社の王さんが取材で日本に来ているので，王さんが出席してくれることになりました。 | 嗯。不过，正好那个时候上海分公司的王先生要来日本采访，所以王先生会出席。 | 「で」：原因/手段；「来ている」：来并停留；「ので」：原因；「くれる」：受益；「ことになりました」：客观结果/决定 |
| それはよかった。何といっても，上海は，君と町子さんが出会った思い出の場所だからね。 | 那太好了。毕竟，上海是你和町子女士相识的、充满回忆的地方嘛。 | 「それはよかった」：那太好了；「何といっても」：固定表达“毕竟”；「出会った」：定语从句；「だからね」：原因+共鸣 |

## 课文 - 日本の人口が減っている——少子化

&emsp;&emsp;世界の人口は増加する一方だが，日本の人口は減り始めている。2005年の統計によると，世界の人口は約65億人で，前の年より約7，550万人も増えている。それに対して，日本の人口は約1億2，775万人で，前の年から約1万人も減少している。

&emsp;&emsp;その原因の1つは，生まれる子供の数が少なくなっていることにある。こうした現象を「少子化」と呼んでいるが，この傾向は今後も続いていくと予測されている。

&emsp;&emsp;「少子化」の背景には，結婚，出産，育児に対する人々の意識の変化がある。例えば，結婚年齢が遅くなる「晩婚化」の傾向は，年々強まっている。結婚時の女性の平均年齢は，1977年には25.0歳であったが，1992年には26.0歳，2004年には27.8歳になった。高年齢になると出産を控える傾向が強まり，1人の女性が出産する子供の数を示す「出生率」が下がる。それが少子化の原因につながっている。

&emsp;&emsp;また，物価の上昇，高額な稅金，住宅問題などのために，結婚をしても子供を作らない夫婦が増えている。自分の意志で結婚しない人も増えており，「非婚率」も上がっている。

&emsp;&emsp;2003年の年間出産数は，1973年の約半分に減った。政府も出生率の低下を防ぐための取り組みを行っている。しかし，人々の意識が簡単に変わるわけはなく，結果が出るまでには，長い年月が必要だろう。

### 课文翻译 - 日本人口在减少——少子化  

&emsp;&emsp;世界人口在不断增长，而日本人口却开始减少。根据2005年的统计，世界人口约为65亿，比前一年增加了约7,550万之多。与此相对，日本人口约为1亿2,775万，比前一年减少了约1万之多。其中一个原因在于，出生的孩子数量正在减少。这种现象被称为“少子化”，并且预测这种趋势今后仍将持续。  

&emsp;&emsp;“少子化”的背景是，人们对结婚、生育、育儿的意识发生了变化。例如，结婚年龄推迟的“晚婚化”趋势逐年增强。结婚时女性的平均年龄，1977年为25.0岁，但到了1992年变为26.0岁，2004年变为27.8岁。随着年龄增高，避免生育的倾向加剧，表示每位女性生育孩子数量的“出生率”下降。这便导致了少子化。  

&emsp;&emsp;另外，因物价上涨、高额税金、住房问题等，即使结婚也不生孩子的夫妇在增加。主动选择不结婚的人也在增加，“非婚率”也在上升。

&emsp;&emsp;2003年的年出生数，减少到了1973年的一半左右。政府也在采取措施以防止出生率下降。但是，人们的意识并不会轻易改变，要见到效果，大概需要长年累月吧。

### 课文语法解析

| 原文 | 翻译 | 语法解析 |
| ------ | ------ | ---------- |
| 世界の人口は増加する一方だが，日本の人口は減り始めている。 | 世界人口在不断增长，而日本人口却开始减少。 | 「一方だが」：对比，“一方面…另一方面…”；「減り始めている」：连用形+始める（开始）+ている（持续状态） |
| 2005年の統計によると，世界の人口は約65億人で，前の年より約7，550万人も増えている。 | 根据2005年的统计，世界人口约为65亿，比前一年增加了约7,550万之多。 | 「によると」：根据；「で」：中顿判断；「より」：比较；「も」：强调数量多；「増えている」：状态 |
| それに対して，日本の人口は約1億2，775万人で，前の年から約1万人も減少している。 | 与此相对，日本人口约为1亿2,775万，比前一年减少了约1万之多。 | 「それに対して」：固定表达，“与此相对”；「から」：时间起点；「も」：强调；「減少している」：状态 |
| その原因の1つは，生まれる子供の数が少なくなっていることにある。 | 其中一个原因在于，出生的孩子数量正在减少。 | 「〜にある」：在于；「少なくなっている」：变少了（状态变化）；「こと」：名词化 |
| こうした現象を「少子化」と呼んでいるが，この傾向は今後も続いていくと予測されている。 | 这种现象被称为“少子化”，并且预测这种趋势今后仍将持续。 | 「と呼んでいる」：称之为；「が」：顺接或铺垫；「続いていく」：持续下去；「と予測されている」：被预测为…（被动+传闻） |
| 「少子化」の背景には，結婚，出産，育児に対する人々の意識の変化がある。 | “少子化”的背景是，人们对结婚、生育、育儿的意识发生了变化。 | 「には」：表存在的地点/范围；「に対する」：对…（的）；「がある」：存在 |
| 例えば，結婚年齢が遅くなる「晩婚化」の傾向は，年々強まっている。 | 例如，结婚年龄推迟的“晚婚化”趋势逐年增强。 | 「例えば」：例如；「遅くなる」：变迟；「年々」：逐年；「強まっている」：正在变强 |
| 結婚時の女性の平均年齢は，1977年には25.0歳であったが，1992年には26.0歳，2004年には27.8歳になった。 | 结婚时女性的平均年龄，1977年为25.0岁，但到了1992年变为26.0岁，2004年变为27.8岁。 | 「には」：时间点强调；「であった」：过去判断（である的过去）；「が」：转折；「になった」：变成 |
| 高年齢になると出産を控える傾向が強まり，1人の女性が出産する子供の数を示す「出生率」が下がる。 | 随着年龄增高，避免生育的倾向加剧，表示每位女性生育孩子数量的“出生率”下降。 | 「〜になると」：一变成…就…；「控える」：控制/避免；「強まり」：连用形中顿；「示す」：定语修饰；「下がる」：下降 |
| それが少子化の原因につながっている。 | 这便导致了少子化。 | 「つながっている」：连接/导致（状态） |

| 2003年の年間出産数は，1973年の約半分に減った。 | 2003年的年出生数，减少到了1973年的一半左右。 | 「〜に減った」：减少到…；「約半分」：约一半 |

| しかし，人々の意識が簡単に変わるわけはなく，結果が出るまでには，長い年月が必要だろう。 | 但是，人们的意识并不会轻易改变，要见到效果，大概需要长年累月吧。 | 「わけはない」：不可能/不会…；「出るまでには」：到出现为止；「必要だろう」：推测需要 |
