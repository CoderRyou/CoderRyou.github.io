## 会话 - トラブル

（向附近的工作人员询问）

**王：** あのう，荷物が出てこないんですけど…。

**手荷物係：** えつ，そうですか。失礼ですが，どのようなお荷物ですか。

**王：** 黒いスーツケースで，「王風」という名札が付けてあります。いくら待っても出てこないんですよ。

**手荷物係：** 申し訳ございません。すぐにお調べします。少々お待ちください。

（另ー边，JC策划公司总部的山田在到港大厅等着迎接王风，但怎么等都不见王风出来）

**山田：** すみません。AJL205便で来る人を待っているんですが，まだ出てこないんです。

**空港職員：** 上海からの便ですね。予定どおり17時30分に到着いたしましたが…。よろしければ，お呼び出しのアナウンスをいたしましょうか。

**山田：** ええ。お願いします。

（在行李提取处，王风的行李还没出来。找王风的广播在机场内响起）

**王：** ちよっと，いったい，いつまで待てばいいんですか。もう6時半ですよ。

**手荷物係：** 申し訳ございません。ただ今お調べしております。もう少しお待ちいただけませんか。

**王：** そんな，困りますよ。ロビーで同僚が待っているんです。早くしてもらえませんか。

**手荷物係：** 本当に申し訳ございません。後ほど，お泊まりのホテルにお届けいたします。よろしいでしょうか。

**王：** （叹息）仕方ないですね。できるだけ早く届けてくださいよ。

### 会话翻译 - 麻烦

（向附近的工作人员询问）

**王：** 那个……我的行李一直没出来……

**行李员：** 诶，是吗？不好意思，请问是什么样的行李？

**王：** 是一个黑色行李箱，上面贴着“王风”的姓名牌。我等了很久都没出来。

**行李员：** 非常抱歉。我马上为您查询。请您稍等片刻。

（另一边，JC策划公司总部的山田在到港大厅等着迎接王风，但怎么等都不见王风出来）

**山田：** 打扰一下。我在等乘坐AJL205航班的人，但他还没出来。

**机场职员：** 是从上海来的航班吧。已经按计划在17点30分抵达了……如果可以的话，我帮您广播找人好吗？

**山田：** 好的。麻烦您了。

（在行李提取处，王风的行李还没出来。找王风的广播在机场内响起）

**王：** 喂，到底还要我等多久啊？已经6点半了。

**行李员：** 实在抱歉。我们正在查询。能否请您再稍等一会儿？

**王：** 这怎么行，我很为难啊。同事在候机厅等着我呢。能不能快点？

**行李员：** 真的非常抱歉。稍后我们会将行李送到您入住的酒店。这样可以吗？

**王：** （叹气）真没办法啊。请尽快送到。

### 会话语法解析

| 原文 | 翻译 | 语法解析 |
| :--- | :--- | :--- |
| **王：** あのう，荷物が出てこないんですけど…。 | **王：** 那个……我的行李一直没出来…… | **1. あのう**<br>**说明：** 语气词，表示犹豫或引起对方注意。<br>**2. ～てこない**<br>**说明：** “～てくる”的否定形，表示动作不朝说话人方向发生。“出てこない”即“不出来”。<br>**3. ～んですけど**<br>**说明：** “～のですけど”的口语形式，用于陈述情况并引出后续请求或问题。 |
| **手荷物係：** えつ，そうですか。失礼ですが，どのようなお荷物ですか。 | **行李员：** 诶，是吗？不好意思，请问是什么样的行李？ | **1. 失礼ですが**<br>**说明：** 固定表达，用于提问或做出某些行为前的礼貌开场白。<br>**2. どのような**<br>**说明：** “什么样的”，比“どんな”更正式、礼貌。 |
| **王：** 黒いスーツケースで，「王風」という名札が付けてあります。 | **王：** 是一个黑色行李箱，上面贴着“王风”的姓名牌。 | **1. ～という名札**<br>**说明：** “叫做……的姓名牌”。<br>**2. ～が付けてあります**<br>**说明：** “付ける”的持续体（てある形式），表示动作结果的状态，“贴有……”。 |
| いくら待っても出てこないんですよ。 | 我等了很久都没出来。 | **1. いくら～ても**<br>**说明：** “无论怎么……也……”，表示让步条件。<br>**2. ～んですよ**<br>**说明：** “～のですよ”的口语形式，表示强调说明或略带不满。 |
| **手荷物係：** 申し訳ございません。すぐにお調べします。少々お待ちください。 | **行李员：** 非常抱歉。我马上为您查询。请您稍等片刻。 | **1. 申し訳ございません**<br>**说明：** “すみません”的非常正式、郑重的道歉用语。<br>**2. お～します**<br>**说明：** 自谦语结构，“お調べします”表示“（我）为您查询”。<br>**3. お～ください**<br>**说明：** 尊敬语结构，“お待ちください”是“待ってください”的尊敬说法。 |
| **山田：** すみません。AJL205便で来る人を待っているんですが，まだ出てこないんです。 | **山田：** 打扰一下。我在等乘坐AJL205航班的人，但他还没出来。 | **1. ～で来る人**<br>**说明：** “乘坐……来的人”。“で”表示交通工具。<br>**2. ～を待っているんですが**<br>**说明：** “正在等……”，后接“が”引出问题或请求。 |
| **空港職員：** 上海からの便ですね。予定どおり17時30分に到着いたしましたが…。 | **机场职员：** 是从上海来的航班吧。已经按计划在17点30分抵达了…… | **1. ～からの便**<br>**说明：** “从……来的航班”。<br>**2. 予定どおり**<br>**说明：** 副词，“按计划”、“照预定”。<br>**3. ～いたしました**<br>**说明：** “する”的自谦语过去式，表示“（航班）抵达了”。 |
| よろしければ，お呼び出しのアナウンスをいたしましょうか。 | 如果可以的话，我帮您广播找人好吗？ | **1. よろしければ**<br>**说明：** “よろしい”的假定形，表示“如果方便的话”。<br>**2. お～いたしましょうか**<br>**说明：** 自谦语结构，表示主动提议为对方做某事，“我为您……吧？”。“お呼び出しのアナウンス”意为“呼叫广播”。 |
| **山田：** ええ。お願いします。 | **山田：** 好的。麻烦您了。 | **语法：** 固定表达。 |
| **王：** ちよっと，いったい，いつまで待てばいいんですか。もう6時半ですよ。 | **王：** 喂，到底还要我等多久啊？已经6点半了。 | **1. ちよっと**<br>**说明：** 此处表示轻微的不满或唤起注意。<br>**2. いったい**<br>**说明：** 副词，与疑问词呼应，加强疑问语气，“到底……”。<br>**3. ～ばいい**<br>**说明：** 表示条件，“只要……就行”。“いつまで待てばいいんですか”意为“要等到什么时候才行呢？”。 |
| **手荷物係：** 申し訳ございません。ただ今お調べしております。もう少しお待ちいただけませんか。 | **行李员：** 实在抱歉。我们正在查询。能否请您再稍等一会儿？ | **1. ただ今**<br>**说明：** 副词，“现在”、“此刻”。<br>**2. お～しております**<br>**说明：** “～ている”的自谦语形式，表示动作正在进行，“正在为您查询”。<br>**3. お～いただけませんか**<br>**说明：** 非常礼貌的请求形式，“能否请您……？”。“いただけませんか”是“いただく”的可能形的否定疑问形式。 |
| **王：** そんな，困りますよ。ロビーで同僚が待っているんです。早くしてもらえませんか。 | **王：** 这怎么行，我很为难啊。同事在候机厅等着我呢。能不能快点？ | **1. 困りますよ**<br>**说明：** “我很为难”，表示不满或请求对方解决问题。<br>**2. ～で待っている**<br>**说明：** “在……等着”。“で”表示地点。<br>**3. ～てもらえませんか**<br>**说明：** “～てもらう”的可能形的否定疑问形式，表示委婉但直接的请求，“能不能请你（为我）……？”。 |
| **手荷物係：** 本当に申し訳ございません。後ほど，お泊まりのホテルにお届けいたします。よろしいでしょうか。 | **行李员：** 真的非常抱歉。稍后我们会将行李送到您入住的酒店。这样可以吗？ | **1. 後ほど**<br>**说明：** 副词，“稍后”、“过一会儿”。<br>**2. お～にお届けいたします**<br>**说明：** 自谦语结构，“（我们）给您送到……”。“お泊まりのホテル”是“您入住的酒店”的尊敬说法。 |
| **王：** （叹息）仕方ないですね。できるだけ早く届けてくださいよ。 | **王：** （叹气）真没办法啊。请尽快送到。 | **1. 仕方ない**<br>**说明：** 固定表达，“没办法”、“不得已”。<br>**2. できるだけ**<br>**说明：** 副词，“尽可能”。<br>**3. ～てくださいよ**<br>**说明：** “よ”加强请求或叮嘱的语气。 |

## 课文 - イタリア旅行

**4月1日** 17時20分，あこがれのローマに到着。荷物を受け取って，入国。旅行社の人が迎えに来ているはずだが，見つからない。もし来なかったらと思い，不安になった時，背の高い男性が，にこにこしながら近づいてきて何か言う。だれかと思ったら，我々を迎えに来たガイドさんだった。

**4月2日** 6時30分起床。今日はローマ市内観光。お土産物屋さんは，わたしたちの顔を見ると，まず，「ニーハオ」と呼びかけ，黙っていると，「アンニョンハシ厶ニカ」。さらに黙っていると，「こんにちは」。海外旅行に来る東洋人は中国，韓国，日本の順になっているのだろうか。

**4月3日** ポンペイ見学。西暦79年に火山の爆発で埋まってしまったローマ時代の都市だ。1900年以上も昔にこんな立派な都市があったなんて，信じられない。

### 课文翻译

**4月1日** 下午5点20分，抵达向往已久的罗马。取完行李，通过入境检查。旅行社的人应该会来接我们，但我没找到。正想着“要是没来怎么办”，感到不安的时候，一位高个子男性微笑着走过来说了些什么。我还在想这是谁，原来就是来接我们的导游。

**4月2日** 早上6点30分起床。今天是罗马市内观光。纪念品店的店员一看到我们的脸，首先喊“你好（中文）”，如果我们不说话，就接着说“安宁哈塞哟（韩文）”。如果我们还是沉默，就换成“こんにちは（日文）”。来海外旅行的东方人难道是按中国、韩国、日本的顺序排列的吗？

**4月3日** 参观庞贝古城。这是公元79年因火山爆发而被掩埋的罗马时代的城市。1900多年前竟然就有这样出色的城市，真是令人难以置信。

| 原文 | 翻译 | 语法解析 |
| :--- | :--- | :--- |

| 今日はローマ市内観光。 | 今天是罗马市内观光。 | **语法：** 名词谓语句，省略判断助动词，日记体常见。 |

| **4月3日** ポンペイ見学。 | **4月3日** 参观庞贝古城。 | **语法：** 名词+サ变动词词干，日记体简洁表达。 |
