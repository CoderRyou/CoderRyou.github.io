会话 - 商品紹介

（首先由龙虎酒业公司的佐藤和大山对“金星”做说明）

佐藤：「金星」が日本で発売されたのは，10年前です。最初はなかなか売れなくて苦労しましたが，3年前，コンテストで最高賞を受賞したのをきっかけにして，売れるようになりました。

**大山：**最近では，海外からの注文も増えてきています。

**佐藤：**ところで，「金星」という名前は，中国人に受けるでしょうか。日本と中国とでは，意味の違う漢字があると聞きましたが。

**王：**この言葉は中国語でも同じ意味ですから，大丈夫ですよ。

**李：**でも，「きんぼし」と読むと，意味が違いますよね。

王：「きんぼし」と「きんせい」は意味が違うんですか。

**佐藤：**ええ，「金星」は星の名前ですが，「金星」というのは，「すばらしい働きをすること」を表します。相撲で位の低い力士が橫綱に勝った時に「金星をあげる」ということからきた書葉です。

**王：**そうですか。相撲からきた言葉ですか。このお酒も金星をあげられるようにしたいですね。

（李秀丽向佐藤提问）

**李：**話は変わりますが，「金星」って，ボトルの形がすてきですね。まるでワインのボトルみたいです。

**佐藤：**ありがとうございます。特にこのデザインは若者に人気があるんですよ。

**王：**このボトルなら，中国でも売れるのは間違いないですね。

**大山：**それを聞いて安心しました。

会话翻译

（首先由龙虎酒业公司的佐藤和大山对“金星”做说明）

佐藤： “金星”在日本发售，是10年前的事了。最初完全卖不动，很是辛苦。但3年前，以在比赛中获得最高奖为契机，变得好卖起来了。

大山： 最近，来自海外的订单也渐渐增加了。

佐藤： 话说回来，“金星”这个名字，能被中国人接受吗？我听说在日本和中国，有些汉字意思不同。

王： 这个词在中文里意思也一样，所以没问题的。

李： 但是，如果读作“きんぼし”，意思就不一样了吧。

王： “きんぼし”和“きんせい”意思不同吗？

佐藤： 是的，“金星”虽然是星星的名字，但“金星（きんぼし）”这个词，表示“做出卓越成绩”的意思。这是源自相扑运动中，地位低的力士战胜横纲时所说的“获得金星”这个词。

王： 原来如此。是来自相扑的词啊。希望这款酒也能“获得金星”（取得成功）呢。

（李秀丽向佐藤提问）

李： 话说回来（换个话题），“金星”这瓶子的形状真不错啊。简直像葡萄酒瓶一样。

佐藤： 谢谢。特别是这个设计在年轻人中很受欢迎。

王： 如果是这个瓶子，在中国也肯定能卖得好。

大山： 听到您这么说我就放心了。

会话语法讲解

原文翻译语法解析
佐藤：「金星」が日本で発売されたのは，10年前です。佐藤： “金星”在日本发售，是10年前的事了。语法： ～が～されたのは～です
说明： “～のは”将“金星が日本で発売された”这个被动句名词化，作整句的主语，后续“です”进行判断。强调事件发生的时间。
最初はなかなか売れなくて苦労しましたが，3年前，コンテストで最高賞を受賞したのをきっかけにして，売れるようになりました。最初完全卖不动，很是辛苦。但3年前，以在比赛中获得最高奖为契机，变得好卖起来了。1. なかなか～なくて
说明： “なかなか”与否定形式呼应，表示“怎么也不……”。“～なくて”是形容词/动词否定形的て形，表示原因或中顿。
2. ～たのをきっかけにして
说明： “～たの”是“こと”的口语形式，将前面从句名词化；“をきっかけにして”表示“以……为契机”。
3. ～ようになる
说明： 表示状态或能力的变化，“变得……起来”。
大山： 最近では，海外からの注文も増えてきています。大山： 最近，来自海外的订单也渐渐增加了。语法： ～てきている
说明： “～てくる”的持续体，表示某种变化从过去开始并持续到现在，有“逐渐……起来”的语感。
佐藤： ところで，「金星」という名前は，中国人に受けるでしょうか。佐藤： 话说回来，“金星”这个名字，能被中国人接受吗？1. ～という名前
说明： “叫做……的名字”，用于说明名称。
2. ～に受ける
说明： 固定搭配，“被……接受、受欢迎”。“に”表示接受的对象。
3. ～でしょうか
说明： “ですか”的礼貌、委婉的说法，表示不确定的询问。
日本と中国とでは，意味の違う漢字があると聞きましたが。我听说在日本和中国，有些汉字意思不同。1. ～とでは
说明： “と”表示并列，“では”表示范围，合起来意为“在……和……（两国）之间/方面”。
2. ～と聞きましたが
说明： 表示传闻，“听说……”。“が”位于句末，表示委婉的语气或引出下文。
王： この言葉は中国語でも同じ意味ですから，大丈夫ですよ。王： 这个词在中文里意思也一样，所以没问题的。语法： ～でも
说明： 提示助词，“でも”接在名词后，表示“即使是在……也”、“在……方面也”。
李： でも，「きんぼし」と読むと，意味が違いますよね。李： 但是，如果读作“kinboshi”，意思就不一样了吧。语法： ～と読むと
说明： “と”表示引用的读法；“～と”接在动词基本形后表示条件，“如果……的话”。
王：「きんぼし」と「きんせい」は意味が違うんですか。王： “kinboshi”和“kinsei”意思不同吗？语法： ～んですか
说明： “のですか”的口语形式，用于寻求确认或解释原因。
佐藤： ええ，「金星」は星の名前ですが，「金星」というのは，「すばらしい働きをすること」を表します。佐藤： 是的，“金星”虽然是星星的名字，但“金星（kinboshi）”这个词，表示“做出卓越成绩”的意思。1. ～が
说明： 表示转折，“虽然……但是”。
2. ～というのは～を表す
说明： “というのは”用于提出主题并加以解释说明；“～を表す”表示“表示……”。
相撲で位の低い力士が橫綱に勝った時に「金星をあげる」ということからきた言葉です。这是源自相扑运动中，地位低的力士战胜横纲时所说的“获得金星”这个词。1. ～で
说明： 表示动作发生的领域或范围，“在……方面”。
2. ～に勝つ
说明： 固定搭配，“战胜……”。“に”表示战胜的对象。
3. ～からきた
说明： 表示来源，“来自于……”。“きた”是“くる”的过去式作定语。
王： そうですか。相撲からきた言葉ですか。王： 原来如此。是来自相扑的词啊。（无特殊语法，重复前文信息表示理解）
このお酒も金星をあげられるようにしたいですね。希望这款酒也能“获得金星”（取得成功）呢。1. ～られる
说明： 可能助动词，“能够获得”。
2. ～ようにしたい
说明： “～ようにする”表示努力使某种状态成立，“したい”是愿望形，合起来表示“希望（努力）使其能够……”。
李： 話は変わりますが，「金星」って，ボトルの形がすてきですね。李： 话说回来（换个话题），“金星”这瓶子的形状真不错啊。1. 話は変わりますが
说明： 转换话题的固定表达，“话说回来”、“换个话题”。
2. ～って
说明： 口语中替代“は”或“というのは”，用于提示主题。
まるでワインのボトルみたいです。简直像葡萄酒瓶一样。语法： まるで～みたいだ
说明： 表示比喻，“简直像……一样”。“まるで”加强比喻的语气。
佐藤： ありがとうございます。特にこのデザインは若者に人気があるんですよ。佐藤： 谢谢。特别是这个设计在年轻人中很受欢迎。语法： ～に人気がある
说明： 固定搭配，“在……中受欢迎”。“に”表示受欢迎的范围或对象。
王： このボトルなら，中国でも売れるのは間違いないですね。王： 如果是这个瓶子，在中国也肯定能卖得好。1. ～なら
说明： 表示假设或限定，“如果是……的话”。
2. ～のは間違いない
说明： “～のは”将“売れる”名词化；“間違いない”是形容词，意为“没错的、肯定的”。
大山： それを聞いて安心しました。大山： 听到您这么说我就放心了。语法： ～て安心する
说明： “て形”表示原因或契机，“安心する”表示放心。

课文 - 日本語の語彙

日本語の単語を，元の言葉は何かという点から分類すると，「和語」「漢語」「外来語」「混種語」の4つに分けることができる。和語はもともと日本語にあった言葉で，漢語は中国語から取り入れられた言葉だ。漢語は「音読み」する。音読みとは，昔の中国語の発音に基づいた読み方だ。外来語は主に19世紀以降，西洋を中心とした外国から取り入れられた言葉で，普通片仮名で書く。さらに，和語，漢語，外来語のうち，2つ以上を組み合わせてできた言葉を混種語と呼ぶ。「消しゴ厶」や「マラソン大会」「正月休み」などの言葉だ。

同じ漢字で表される言葉でも，漢語か和語かで意味が違うことがある。例えば「生物」という言葉は，漢語として「せいぶつ」と読むと，動物や植物の総称となるが，和語として「なまもの」と読むと，煮たり焼いたりしていない食べ物という意昧になる。だから「生物を食べる」という文を「せいぶつをたべる」と読むと，意味がよく分からなくなってしまう。

漢語，和語に外来語が加わると，さらに意味の違いが出てくる。例えば，宿泊施設をいう場合，「旅館」というと，たいていの人は，畳の上に布団を敷いて寝るような部屋を想像する。一方，「ホテル」というと，ベッドで寝る部屋を想像することが多い。

课文翻译

日语的单词，根据其源自何种语言这一点来分类的话，可以分为“和语”、“汉语”、“外来语”、“混种语”四种。和语是原本就存在于日语中的词，汉语是从中文引进的词。汉语用“音读”来读。所谓音读，就是基于古汉语发音的读法。外来语主要是19世纪以后，从以西洋为中心的外国引进的词语，通常用片假名书写。此外，将和语、汉语、外来语中的两种以上组合而成的词语，称为混种语。比如“消しゴム（橡皮）”、“マラソン大会（马拉松大会）”、“正月休み（新年假期）”等词。

即使是同一个汉字表示的词，根据它是汉语还是和语，意思有时也会不同。例如“生物”这个词，作为汉语读作“せいぶつ”时，是动植物的总称；但作为和语读作“なまもの”时，就成了未经煮烧的食物（生鲜食品）的意思。因此，如果把“生物を食べる”这个句子读作“せいぶつをたべる”，意思就变得难以理解了。

当汉语、和语再加上外来语时，意思的差别就更明显了。例如，在说住宿设施时，如果提到“旅馆”，大多数人会想象在榻榻米上铺被褥睡觉的房间。另一方面，如果提到“酒店”，则多数人会想象有床的房间。

2. ～ことができる
说明： 表示可能，“能够……”。

说明： “ある”的过去式作定语，表示“曾经存在的”、“原有的”。
